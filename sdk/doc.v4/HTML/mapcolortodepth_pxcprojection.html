Description

The MapColorToDepth function maps an array of color coordinates to depth coordinates.

If the function fails to map certain coordinates, for example, some color value is floored or saturated, the function returns (-1,-1) in the depth coordinates to indicate the failure.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

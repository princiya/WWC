Description

The StreamOption enumerator itemizes supported stream options. You can use bit-OR to specify multiple options.

Name/Description

 

STREAM_OPTION_ANY

No option specified.

 

STREAM_OPTION_DEPTH_PRECALCULATE_UVMAP

A performance hint to anticipate using UV map in subsequent operations.

 

STREAM_OPTION_STRONG_STREAM_SYNC

A hint to use hardware-based strong stream synchronization. The device synchronizes all streams with this flag.

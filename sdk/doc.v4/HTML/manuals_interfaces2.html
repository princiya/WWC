The PXC[M]FaceData interface exposes the following nested interfaces:

Interface

Description

Face

The aggregated interface that provides access to the below data instances.

DetectionData

The interface to access the face location parameters.

LandmarksData

The interface to access the face landmark parameters.

PoseData

The interface to access the face pose parameters.

ExpressionsData

The interface to access the facial expression parameters.

RecognitionData

The interface to access the face recognition parameters.

RecognitionModuleData

The interface to manage the face recognition database.

Description

The PXC[M]Capture interface provides member functions to query video capture devices and create an instance of the capture device.

The application can use the QueryImpl function to enumerate and locate the desired capture module, and the CreateImpl function to create an instance of the PXC[M]Capture interface.

All capture modules have the following module descriptions:        

C++

group=PXCSession::IMPL_GROUP_SENSOR;

subgroup=PXCSession::IMPL_SUBGROUP_VIDEO_CAPTURE

 

C#

group=PXCMSession.ImplGroup.IMPL_GROUP_SENSOR;

subgroup=PXCMSession.ImplSubgroup.IMPL_SUBGROUP_VIDEO_CAPTURE;

 

Java

group=PXCMSession.ImplGroup.IMPL_GROUP_SENSOR;

subgroup=PXCMSession.ImplSubgroup.IMPL_SUBGROUP_VIDEO_CAPTURE;

Example

Example 129 shows how to create a capture object.

C++ Example 129: Enumerate Video Capture Modules

PXCSession::ImplDesc desc={};

desc.group=PXCSession::IMPL_GROUP_SENSOR;

desc.subgroup=PXCSession::IMPL_SUBGROUP_VIDEO_CAPTURE;

 

for (int i=0;;i++) {

   PXCSession::ImplDesc desc1;

   if (session->QueryImpl(&desc,i,&desc1)<PXC_STATUS_NO_ERROR) break;

 

   PXCCapture *capture;

   session->CreateImpl<PXCCapture>(&desc1,&capture);

   ....

   capture->Release();

}

C# Example 129: Enumerate Video Capture Modules

PXCMSession.ImplDesc desc=new PXCMSession.ImplDesc() {

   group=PXCMSession.ImplGroup.IMPL_GROUP_SENSOR,

   subgroup=PXCMSession.ImplSubgroup.IMPL_SUBGROUP_VIDEO_CAPTURE,

};

 

for (uint i=0;;i++) {

   PXCMSession.ImplDesc desc1;

   if (session.QueryImpl(desc,i,out desc1)<pxcmStatus.PXCM_STATUS_NO_ERROR) break;

 

   PXCMCapture capture;

   session->CreateImpl<PXCMCapture>(desc1, out capture);

   ....

   capture.Dispose();

}

   ....

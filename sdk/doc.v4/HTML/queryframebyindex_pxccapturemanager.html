Description

The QueryFrameByIndex function returns the frame index of the frame closest to the current playback position. The frame index is zero-based.

Return Status

 

The frame index, or -1 if the playback position is at the end of the file.

Remarks

Unity Web

File recording and playback are not supported in Unity Web. This function is not supported.

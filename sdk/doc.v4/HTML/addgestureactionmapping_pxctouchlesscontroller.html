Parameters

 

gesture

The gesture name.

 

action

The OS injection action.

 

handler

The optional callback function instance.

Description

The AddGestureActionMapping function adds a mapping between the gesture recognition event and an OS injection action. If the handler is specified, the function calls back the handler when the gesture is triggered.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_DATA_NOT_INITIALIZED

The feature is not initialized.

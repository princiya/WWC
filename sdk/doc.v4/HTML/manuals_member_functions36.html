The PXC[M]HandData interface exposes the following member functions:

Feature

Member Function

Description

Alert Handling

QueryFiredAlertData

Return the fired alert of a given index.

QueryFiredAlertsNumber

Return the number of fired alert events.

IsAlertFired

Check if an alert is fired.

IsAlertFiredByHand

Check if an alert is fired by the hand.

Hand Tracking

QueryNumberOfHands

Return the number of detected hands.

QueryHandData

Return the detected hand data.

QueryHandDataById

Return the detected hand data from the hand identifier.

QueryHandId

Return the detected hand identifier.

Gesture Recognition

QueryFiredGestureData

Return the fired gesture data.

QueryFiredGesturesNumber

Return the number of fired gestures.

IsGestureFired

Check if a gesture is fired.

IsGestureFiredByHand

Check if a gesture is fired by the hand.

Configuration

Update

Update the PXC[M]HandData instance to be latest available output.

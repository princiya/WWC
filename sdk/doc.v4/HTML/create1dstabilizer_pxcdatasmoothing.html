Parameters

 

strength

The strength value. The default is 0.5 if not specified.

 

radius

The radius value. The default is 0.03 if not specified.

Description

The Create1DStabilizer function creates an instance of the 1D stabilizer smoothing algorithm.

The application must release the instance after use.

Return Status

 

 

The Smoother1D instance, or NULL/null if there is any error.

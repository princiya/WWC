Description

The ProfileInfo structure describes the text to speech module configurations.

Members

 

language

The language of the text to speech module. See the LanguageType enumerator for definitions.

 

voice

The voice of the text to speech module. See the VoiceType enumerator for definitions.

 

outputs

Output audio format description; see the AudioInfo structure for details.

 

rate

Set the speaking rate. The value is between 50 (half the default rate) and 400 (four times the default rate). The default value is 100.

 

volume

Set the volume level. The value is between 0 (mute) and 100 (the maximum volume). The default value is 80.

 

pitch

Scale the voice pitch with a factor between 50 (half the defult pitch) and 200 (two times the default pitch). The default value is 100.

 

eosPauseDuration

Set the end of sentence pause duration to a value between 0 and 9. The pause is about 200 milliseconds multiplied by the value.

PXCAudioSourceService::ReadSample

Syntax

C++

pxcStatus ReadSample(PXCAudio **sample);

 

Parameters

 

audio

The audio sample instance, to be returned.

Description

The ReadSample function waits until the next audio sample is ready and returns it.

The application must release the sample instance.

Return Status

 

PXC_STATUS_NO_ERROR

The function returned successfully.

 

PXC_STATUS_DEVICE_LOST

The audio device is disconnected.

 

PXC_STATUS_ITEM_UNAVAILABLE

The audio stream is ended.

 

PXC_STATUS_TIME_GAP

There is a jump in the time stamp, compared with the most recent audio sample.

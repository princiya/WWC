PXCMFaceModule

Description

The PXC[M]FaceModule interface provides member functions to perform face tracking and detection.

The application can use the EnableFace function to activate face tracking in the PXC[M]SenseManager pipeline.

Example 145 shows how to perform face tracking with a live camera streaming.

C++ Example 145: Face Tracking with Live Streaming

// Create a SenseManager instance

PXCSenseManager *sm=session->CreateSenseManager();

 

// Enable face tracking in the pipeline

sm->EnableFace();

 

// Get a face instance here (or inside the AcquireFrame/ReleaseFrame loop) for querying capabilities.

PXCFaceModule *face=sm->QueryFace();

...

 

// Initialize the pipeline

sm->Init();

 

// Stream data

while (sm->AcquireFrame(true)>=PXC_STATUS_NO_ERROR) {

  /* retrieve face analysis results */

  face->...

  ...

 

  sm->ReleaseFrame();

}

 

// Clean up

sm->Release();

Parameters

 

data

The mesh data storage allocated by the CreatePXC[M]BlockMeshingData function.

 

bFillHoles

An optional boolean value to indicate whether the function should fill the holes for the mesh data. If omitted, the default is false.

 

options

Additional options for updating the meshes. See the MeshingUpdateInfo structure for details.

Description

The DoMeshingUpdate function performs meshing and hole filling and stores the result mesh data to the mesh data storage.

Run this function in a thread as meshing and hole filling is a lengthy process.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

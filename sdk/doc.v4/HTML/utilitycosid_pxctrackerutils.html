PXCMTrackerUtils.UtilityCosID

Description

The UtilityCosID defines special tracking identifiers used during map creation. You can use it to retrieve the current pose information.

Name/Description

 

CALIBRATION_MARKER

The pose of the detected calibration marker.

 

ALIGNMENT_MARKER

The new pose of the tracked object based on the alignment marker.

 

IN_PROGRESS_MAP

The current pose of the new map as it is created.

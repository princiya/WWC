Limitations

You can use the same C# interfaces described in this manual in the Unity C# webplayer applications, with the following limitations:

•Image data access is limited to low resolutions such as 320x240. Accessing image data may significantly impact application performance. There is no limitation however to access image instance information such as format, resolution, and time stamp.
•Due to latency or security restrictions, certain SDK functions do not support the Unity web player platform. These functions have a respective note in their function Remark sections. The functions include:
oAdvanced programming directly through PXCMCaptureManager is not supported. It is recommended to use the PXCMSenseManager interface.
oAny SDK functions that need to access any files on the local or remote disk.
•Support of event callbacks is limited. The event callbacks are implemented as a one-way function call from the local proxy server to the Unity application. As the local server runs asynchronously to the browser context, the application can only use the event callback to set some application state. It is impossible to access any module data or image content within an event callback.

Example

Example 113 shows a hand tracking Unity script. The script uses the PXCMSenseManager interface to active hand tracking, waits until any hand tracking data is ready and then retrieves it.

public class SenseInput : MonoBehaviour {

   private PXCMSenseManager sm=null;

 

   // Use this for initialization

   void Start () {

       /* Initialize a PXCMSenseManager instance */

       sm = PXCMSenseManager.CreateInstance();

       if (sm==null) return;

 

       /* Enable hand tracking */

       pxcmStatus sts=sm.EnableHand();

         

       /* Initialize the execution pipeline */

       sts=sm.Init();

   }

 

   // Update is called once per frame

   void Update () {

       if (sm==null) return;

 

       /* Wait until any frame data is available */

       if (sm.AcquireFrame(false,0)<pxcmStatus.PXCM_STATUS_NO_ERROR) return;

 

       /* Retrieve hand tracking data if ready */

       PXCMHandModule hand=sm.QueryHand();

       if (hand != null)

       {

           PXCMHandData data = hand.CreateOutput();

           if (data != null)

           {

               data.Update();

               // access hand data

               ....

               data.Dispose();

           }

       }

 

       // Access hand image

       PXCMCapture.Sample sample = sm.QueryHandSample();

 

       if (sample != null)

       {

           if (sample.depth != null) {

               // Access the hand image information

               ....

           }

       }

   }

 

   void OnDisable() {

       if (sm==null) return;

       sm.Dispose();

       sm=null;

   }

}

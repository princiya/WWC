Namespace Hierarchy

JavaScript

PXCMSpeechRecognition.StartRec

Syntax

JavaScript

function StartRec(onRecognition, onAlert);

Parameters

 

onRecognition

The callback function OnRecognition if any recognition event occurs.

 

onAlert

The callback function OnAlert if any alert event occurs.

Description

The StartRec function starts speech recognition.

Return Status

 

The returned promise object contains:

 

sts

The function status code, as defined in the pxcmStatus enumerator.

Remarks

JavaScript

The OnRecognition(data) callback function is invoked when there are any speech recognized.  The fields of the RecognitionData structure are passed at input. See also OnRecognition.

The OnAlert(data) callback function is invoked when there is any alert. The fields of the AlertData structure are passed at input. See also OnAlert.

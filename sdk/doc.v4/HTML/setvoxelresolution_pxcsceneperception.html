Parameters

 

resolution

The voxel resolution enumeration; see the VoxelResolution enumerator for definitions.

Description

The SetVoxelResolution function sets the volume resolution. Use the function before the PXC[M]SenseManager Init function (or the PXC[M]VideoModule SetCaptureProfile function in advanced programming). The resolution is locked afterward.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_ITEM_UNAVAILABLE

The function is called after the Init function.

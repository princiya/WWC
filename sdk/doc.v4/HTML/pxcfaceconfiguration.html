Description

The PXC[M]FaceConfiguration interface provides member functions to configure the face tracking module.

The application can use the CreateActiveConfiguration function to create an instance of this interface.

Data Members

 

detection

Access to the configuration parameters of face detection.

 

landmarks

Access to the configuration parameters of landmark detection.

 

pose

Access to the configuration parameters of pose detection.

 

strategy

The face tracking mode.

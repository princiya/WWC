PXCAudioSourceService::Open

Syntax

C++

pxcStatus Open(AudioProfile *profile);

Parameters

 

profile

The audio device configuration, as described in the AudioProfile structure.

Description

The Open function opens the audio device for streaming.

Return Status

 

PXC_STATUS_NO_ERROR

The function returned successfully.

 

PXC_STATUS_DEVICE_LOST

The audio device is disconnected.

 

PXC_STATUS_PARAM_UNSUPPORTED

There are invalid parameters in the configuration.

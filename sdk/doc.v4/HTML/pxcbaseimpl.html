PXCBaseImpl

Description

The PXCBaseImpl template class contains the default implementation of the PXCBase interface. The application that implements any PXCBase derived interface, such as a callback handler, should derive the implementation from this template class. While PXCBaseImpl template assumes single inheritance, its variations, PXCBaseImpl2 and PXCBaseImpl3, implement two or three inheritances, respectively.

Example

C++ Example 157: Implement an SDK Interface

// This is a stub implementation

 

class AudioSourceImpl: public PXCBaseImpl<PXCAudioSource> {

public:

   virtual void PXCAPI ScanDevices(void) {}

   virtual pxcI32 PXCAPI QueryDeviceNum(void) { return 0; }

   virtual pxcStatus PXCAPI QueryDeviceInfo(pxcI32 didx, DeviceInfo *dinfo) { return PXC_STATUS_FEATURE_UNSUPPORTED; }

   virtual pxcStatus PXCAPI SetDevice(DeviceInfo *dinfo) { return PXC_STATUS_FEATURE_UNSUPPORTED; }

   virtual pxcStatus PXCAPI SetVolume(pxcF32 volume) { return PXC_STATUS_FEATURE_UNSUPPORTED; }

};

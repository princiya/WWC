Parameters

 

photo

The photo image instance.

 

focus

The point of the focusing point (with the color image.)

 

aperture

The size of the blurring area. If omitted, the default is 50.0F.

 

blurType

The blurring algorithm used; see the BlurType enumerator for definitions. If omitted, the default is BLUE_EXPONENTIAL.

Description

The DepthRefocus function creates a refocused photo image with the specified focusing point and blurring the rest area.

The function creates a new photo instance at output. You must release the photo instance after use.

Return Status

 

The refocused photo image instance, or NULL/null if there is any error.

The ExpressionsConfiguration interface exposes the following member functions:

Member Function

Description

Enable

Enable the expression detection.

Disable

Disable the expression detection.

IsEnabled

Get the expression detection status.

EnableAllExpressions

Enable all expressions.

DisableAllExpressions

Disable all expressions.

EnableExpression

Enable a specific expression.

DisableExpression

Disable a specific expression.

IsExpressionEnabled

Check if a specific expression is enabled for detection.

PXCMSenseManager.Handler

Description

The Handler interface is a set of callback functions/delegates to be used during the PXC[M]SenseManager initialization and frame looping operations. The application must implement the interface.

Examples

Example 138 shows how to create a handler.

C++ Example 138: Create a Handler

class MyHandler: public PXCSenseManager::Handler {

public:

 

   virtual pxcStatus PXCAPI OnConnect(PXCCapture::Device *device, pxcBool connected) {

       return PXC_STATUS_DEVICE_FAILED;

   }

 

   virtual pxcStatus PXCAPI OnModuleProcessFrame(pxcUID mid, PXCBase* object) {

       return PXC_STATUS_NO_ERROR;

   }

};

// sm is a PXCSenseManager instance.

MyHandler handler;

sm->Init(&handler);

C# Example 138: Create a Handler

static pxcmStatus OnConnect(PXCMCapture.Device device, bool connected) {

   return pxcmStatus.PXCM_STATUS_DEVICE_FAILED;

}

 

static pxcmStatus OnModuleProcessFrame(int mid, PXCMBase object) {

   return pxcmStatus.PXCM_STATUS_NO_ERROR;

}

 

// sm is a PXCMSenseManager instance

PXCMSenseManager.Handler handler=new PXCMSenseManager.Handler();

handler.onConnect=OnConnect;

handler.onModuleProcessFrame=OnModuleProcessFrame;

sm.Init(handler);

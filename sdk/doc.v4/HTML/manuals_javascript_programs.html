The SDK supports limited modules when programming with JavaScript in a browser context. The list of supported modules is as follows:

•Face detection, landmark tracking, pose and expression detection.
•Hand tracking and gesture recognition.
•Speech recognition (dictation, command and control).

See the sub-sections below for examples of how to program the supported modules.

JavaScript Programming Specifics

The SDK exposes the JavaScript interfaces and methods similar to other supported languages such as C++, C# and Java* with the following differences:

•The SDK exposes static functions by concatenating the name space hierarchy with the underscore character.

PXCMSession_CreateInstance(...);

PXCMSenseManager_CreateInstance(...);
•The SDK exposes constant values or enumerator values in the global object pxcmConst. The name space hierarchy is preserved.

pxcmConst.PXCM_STATUS_NO_ERROR

pxcmConst.PXCMSpeechRecognition.CUID

pxcmConst.PXCMCapture.Device.MIRROR_MODE_HORIZONTAL

pxcmConst.PXCMFaceConfiguration.FACE_MODE_COLOR_PLUS_DEPTH
•Each SDK function returns a promise object. You must use the then function to concatenate several procedural calls.

var sm;

var face;

 

PXCMSenseManager_CreateInstance().then(function (instance) {

   sm = instance;

   return sm.EnableFace(onFaceData);

}).then(function (instance) {

   face=instance;

   return sm.Init();

}).then(function (result) {

   return face.CreateActiveConfiguration();

}).then(function (config) {

   ...

}).then(function (result) {

   return face.ApplyChanges();

}).then(function (result) {

   return sense.StreamFrames();

}).then(function (result) {

}).catch(function (error) {

});
•The SDK redesigns the callback functions to return an aggregation of all module processing data. For example, the hand tracking callback function OnHandData is invoked with all aggregated hand tracking data. This is significantly different from other language programming, where you need to query the specific hand data within the callback function.

function onHandData(mid, module, data) {

   var joints = data.hands[0].trackedJoint;

}
•The SDK supports the limited functions of the following interfaces only:

PXCMBase

A general interface for any unsupported modules. This is reserved.

PXCMCaptureManager

Retrieve information about the device and streams.

PXCMSenseManager

Pipeline control for the face and hand tracking modules.

PXCMSession

The SDK session management interface.

PXCMFaceModule

PXCMFaceConfiguration

The face tracking interface.

PXCMHandModule

PXCMHandConfiguration

The hand tracing interface.

PXCMSpeechRecognition

The speech recognition interface.

Java

pxcmStatus EnableModule(int mid, ImplDesc mdesc);

 

Parameters

 

mid

The module identifier. This is usually the interface identifier.

 

mdesc

The module descriptor to identify the processing module; see the ImplDesc structure for details.

Description

The EnableModule function creates an instance of the specified processing module and inserts it to the processing pipeline.

This is the base function for inserting any modules. The application should use the interface specific helper functions such as EnableHand and EnableFace.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The module instance is created successfully.

Parameters

 

sample

The scene image sample required by the scene perception module. Use the QueryScenePerceptionSample function to retrieve the sample instance.

 

minPlaneArea

Specify the minimum plane area in mm2.

 

maxPlaneNum

Specify the plane number.

 

planeEq

The float array of the plane equations, to be returned. A maximum of 5 planes can be detected. Each row contains the coefficients of the detected plane. E.g., row 0 contains the plane equation: ax+by+cz+w in the form of {a,b,c,w}. If no plane is detected, the row contains zero.

 

PlaneIndex

The array of indexes of size 320x240 to be returned. Each index is the detected plane index, starting 1 (the first plane). If no plane is detected, the index is zero.

Description

The ExtractPlanes function returns information about the detected planar surfaces in the scene.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

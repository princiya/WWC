Description

The ImageData structure describes image storage details.

Members

 

format

The pixel format of image samples. See the PixelFormat enumerator for definitions.

 

planes

Image plane buffers. Compacted or compressed images use planes [0] to point to a single continuous buffer.

 

pitches

Pitches of the plane buffers.

Remarks

C#

The planes field points to array of pointers to the image buffers in the unmanaged memory. The application can use marhsal copy or the provided member functions to access to the image buffers.

Java

The application must use the provided member functions to access to the image buffers.

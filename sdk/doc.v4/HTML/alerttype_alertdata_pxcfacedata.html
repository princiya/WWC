Description

The AlertType enumerator itemizes the alert events.

Name/Description

 

ALERT_NEW_FACE_DETECTED

A new face is detected.

 

ALERT_FACE_NOT_DETECTED

There is no face in the scene.

 

ALERT_FACE_OUT_OF_FOV

The face is out of camera field of view.

 

ALERT_FACE_BACK_TO_FOV

The face is back to field of view.

 

ALERT_FACE_OCCLUDED

The face is occluded.

 

ALERT_FACE_NO_LONGER_OCCLUDED

The face is not occluded.

 

ALERT_FACE_ATTACHED_OBJECT

The face is occluded by some object, for example, a hand.

 

ALERT_FACE_OBJECT_NO_LONGER_ATTACHED

The face is not occluded with the object.

 

ALERT_FACE_LOST

The face is lost tracking.

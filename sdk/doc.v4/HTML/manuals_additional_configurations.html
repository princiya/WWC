Configuring Synthesis Quality

 Top  Previous  Next

You can fine-tune speech synthesis quality as follows:

Initialization Parameters

You can use the following parameters during initialization (SetProfile) to fine-tune speech synthesis. They applies to the complete sentence:

•volume: Set the volume level. The value is between 0 (mute) and 100 (the maximum volume). The default value is 80.
•pitch: Scale the voice pitch with a factor between 50 (half the default pitch) and 200 (two times the default pitch). The default value is 100.
•rate: Set the speaking rate. The value is between 50 (half the default rate) and 400 (four times the default rate). The default value is 100.
•eosPauseDuration: Set the end of sentence pause duration to a value between 0 and 9. The pause is about 200 milliseconds multiplied by the value.

Escape Control Codes

You can insert escape control codes into the speech text to fine-tune part of speech synthesis. The control sequence leads with an <ESC> ASCII code and commands denoted by back slashes.

•Audio recording:

Example: Record after the beep. <ESC>\audio=”c:\beep.wav”\

The control sequence <ESC>\audio=”<file>”\ inserts audio recording into the synthesized speech. The audio file must be a WAV file, encoded with 16-bit PCM and the sample rate matching the speech synthesis setting.

•Pause:

Example: I am <ESC>\pause=300\ John.

The control sequence <ESC>\pause=<time>\ inserts a pause into the synthesized speech. The pause is in milliseconds.

•Text Formatting:

Example: The address reads <ESC>\tn=address\ 2111 NE 25th Ave, Hillsboro, OR <ESC>\tn=normal\

The control sequence <ESC>\tn=<type>\ guides the synthesizer on the text format, as follows:

spell

Spell out the text that follows.

address

Synthesize the text as an address.

sms

Synthesize the text as an SMS message.

normal

Reset to regular text processing.
•Read Mode:

Example: Follow me to spell the word Bee <ESC>\readmode=char\ Bee

The control sequence <ESC>\readmode=<mode>\ changes how the text should be synthesized as follows:

sent

Read the text as a sentence, and pause after each sentence. This is the default.

char

Read the text character by character (similar to the spell text formatting.)

word

Read the text word by word, with pause between words.

line

Read the text line by line, with pause between lines.

explicit_eos

Break the sentence only when seeing the <ESC>\eos=<value>\ control sequence.

•Word Prominence Level

Example: I <ESC>\nlu=PRM:3\ am the person you are looking for.

The control sequence <ESC>\nlu=PRM:<level>\ adjusts the word prominence level as follows:

0

Reduced.

1

Stressed.

2

Accented.

3

Emphasized.

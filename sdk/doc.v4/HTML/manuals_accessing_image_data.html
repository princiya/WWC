Accessing Image Data

 Top  Previous  Next

If segmentation is enabled, you can retrieve the segmentation image for the hand. This is a black-and-white image, in which the hand pixels are white and all other pixels are black.

The IHand object contains the following methods for retrieving image-related data:

•HasSegmentationImage - returns true if segmentation data exists, otherwise returns false.
•QuerySegmentationImage – returns the segmentation image.

Speech Recognition and Synthesis

 Top  Previous  Next

The SDK speech modules provide command and control, dictation, and text to speech capabilities.

Speech recognition translates input speech into text. There are two modes of operations:

•Command and control: The application defines a list of context words as the command list and the SDK module recognizes speech based only on the command list.
•Dictation: The SDK module returns the most likely dictated sentence.

Text to speech is also called speech synthesis. It translates text back to speech.

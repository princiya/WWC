Description

The QueryDepthSensorRange function returns the depth sensor sensing distance, in mm.

The property value is the model fixed value, and not the device instance calibrated value.

Return Status

 

The depth sensor sensing distance values. If any error, the structure is filled with PROPERTY_VALUE_INVALID.

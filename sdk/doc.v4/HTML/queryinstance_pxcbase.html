Parameters

 

cuid

The interface identifier. For any SDK interface X, the interface identifier is X::CUID (C++) or X.CUID (C#,Java).

Description

The QueryInstance function returns the instance of the requested interface if the implementation supports such interface, or NULL/null if the implementation does not support the requested interface.

The application should not release the returned instance, which is internally managed.

Return Status

 

None

 

Remarks

C++

For simplicity, the application can use the QueryInstance<T> template. For example, the following statement casts to a Y interface from an X interface: Y *y=x->QueryInstance<Y>();

C#

For simplicity, the application can use the QueryInstance<T> template. For example, the following statement casts to a Y interface from an X interface: Y y=x.QueryInstance<Y>();

Example

Example 127 shows how to retrieve the PXC[M]Metadata interface from the PXC[M]Session instance.

C++ Example 127: Retrieve the PXCMetadata interface from PXCSession

PXCMetadata *meta_data=session->QueryInstance<PXCMetadata>();

C# Example 127: Retrieve the PXCMMetadata Interface from PXCMSession

PXCMMetadata meta_data=session.QueryInstance<PXCMMetadata>();

Java Example 126: Retrieve the PXCMMetadata Interface from PXCMSession

PXCMMetadata meta_data=(PXCMMetadata)session.QueryInstance(PXCMMetadata.CUID);

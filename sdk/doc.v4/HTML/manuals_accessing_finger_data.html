Accessing Finger Data

 Top  Previous  Next

You can query for information about each of the five fingers. This includes the finger’s “foldedness” (a numeric value indicating the degree that a finger is bent as opposed to straight), and the radius (in meters) of the tracked fingertip.

The IHand object contains the following method for retrieving finger data:

•QueryFingerData – returns the finger’s “foldedness” value and fingertip radius. Note that the radius depends on calibration data, thus requires the calibration process to be completed.

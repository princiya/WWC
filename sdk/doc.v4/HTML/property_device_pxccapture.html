PXCMCapture.Device.Property

Description

The Property enumerator itemizes supported device properties, for advanced usages, where the device property values are negotiated between the I/O device and the algorithm modules during the pipeline initialization.

In simple use cases, the application can use the helper functions QueryXXX and SetXXX of the Device interface to get/set device property values, without involving the Property enumerator.

Name/Description

 

Device Common Properties

 

PROPERTY_DEVICE_ALLOW_PROFILE_CHANGE

If set to true, the device returns PXC_STATUS_STREAM_CONFIG_CHANGED when the stream configuration is changed.

 

PROPERTY_DEVICE_MIRROR

The orientation of the camera images.. See the MirrorMode enumerator for definitions.

 

Color Stream Properties

 

 

PROPERTY_COLOR_EXPOSURE

The color camera exposure, in log base 2 seconds.

 

PROPERTY_COLOR_BRIGHTNESS

The color camera brightness from  -10,000 (pure black) to 10,000 (pure white).

 

PROPERTY_COLOR_CONTRAST

The color camera contrast, from 0 to 10,000.

 

PROPERTY_COLOR_SATURATION

The color camera saturation, from 0 to 10,000.

 

PROPERTY_COLOR_HUE

The color camera hue, from -18,000 to 18,000 (representing -180 to 180 degrees.)

 

PROPERTY_COLOR_GAMMA

The color camera gamma, from 1 to 500.

 

PROPERTY_COLOR_WHITE_BALANCE

The color camera balance, as a color temperature in degrees Kelvin.

 

PROPERTY_COLOR_SHARPNESS

The color camera sharpness, from 0 to 100.

 

PROPERTY_COLOR_GAIN

The color camera gain adjustment, with negative values darker, positive values brighter, and zero as normal.

 

PROPERTY_COLOR_BACK_LIGHT_COMPENSATION

The color camera back light compensation, with 1 to turn on and 0 to turn off.

 

PROPERTY_COLOR_FIELD_OF_VIEW

The color sensor horizontal and vertical field of view parameters, in degrees, as defined in the PXC[M]PointF32 structure.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_COLOR_FOCAL_LENGTH

The color sensor focal length in pixels along the x and y axes, as defined in the PXC[M]PointF32 structure. The parameters vary with the color stream resolution setting.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_COLOR_FOCAL_LENGTH_MM

The color sensor focal length in millimeters.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_COLOR_PRINCIPAL_POINT

The color sensor principal point in pixels along the x and y axes, as defined in the PXC[M]PointF32 structure. The parameters vary with the color stream resolution setting.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_COLOR_POWER_LINE_FREQUENCY

The power line frequency. See the PowerLineFrequency enumerator for definitions.

 

Depth Stream Properties

 

 

PROPERTY_DEPTH_LOW_CONFIDENCE_VALUE

The special depth map value to indicate that the corresponding depth map pixel is of low-confidence. The following are possible reasons for low-confidence:

▪Objects are too close to the camera.
▪Objects are too far away from the camera.
▪Objects have low infrared reflectivity.

 

PROPERTY_DEPTH_CONFIDENCE_THRESHOLD

The confidence threshold that is used to floor the depth map values.

 

PROPERTY_DEPTH_UNIT

The precision of native depth pixel values in micrometre, or μm.

 

PROPERTY_DEPTH_FIELD_OF_VIEW

The depth-sensor horizontal and vertical field of view parameters, in degrees, as defined in the PXC[M]PointF32 structure.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_DEPTH_SENSOR_RANGE

The depth-sensor, sensing distance parameters, in millimeters, as defined in the PXC[M]RangeF32 structure.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_DEPTH_FOCAL_LENGTH

The depth-sensor focal length in pixels along the x and y axes, as defined in the PXC[M]PointF32 structure. The parameters vary with the depth stream resolution setting.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_DEPTH_FOCAL_LENGTH_MM

The depth-sensor focal length in millimeters.

The property value is the model fixed value, and not the device instance calibrated value.

 

PROPERTY_DEPTH_PRINCIPAL_POINT

The depth-sensor principal point in pixels along the x and y axes, as defined in the PXC[M]PointF32 structure. The parameters vary with the depth stream resolution setting.

The property value is the model fixed value, and not the device instance calibrated value.

 

Intel® RealSense™ 3D Camera: Front Facing Camera Model F200

 

PROPERTY_IVCAM_ACCURACY

The projected pattern. See the IVCAMAccuracy for definitions.

 

PROPERTY_IVCAM_FILTER_OPTION

The smoothing aggressiveness parameter. See Table 15 for definitions.

 

PROPERTY_IVCAM_LASER_POWER

The power of laser projector in the range of 0 (min power) to 16 (max power).

 

PROEPRTY_IVCAM_MOTION_RANGE_TRADE_OFF

The motion vs. range trade off option. The value is in the range of 0 (short exposure, short range, and better motion) to 100 (long exposure and long range.)

 

Intel® RealSense™ 3D Camera: Rear Facing Camera Model R200

 

PROPERTY_DS_COLOR_RECTIFICATION

Boolean: The color image rectification mode. The value is read only.

 

PROPERTY_DS_CROP

Boolean: The left and right image cropping mode. If true, the left/right images are cropped to match the resolution of the depth image.

 

PROPERTY_DS_DEPTH_RECTIFICATION

Booelan: The depth image rectification mode. The value is read only.

 

PROPERTY_DS_EMITTER

Boolean: The infrared emitter status.

 

PROPERTY_DS_DISPARITY_MULTIPLIER

The disparity scale factor in the disparity output mode. The default value is 32.

 

PROPERTY_DS_DISPARITY_OUTPUT

Boolean: The disparity output mode. If true, set the output is the inverse distance. Otherwise, the output is the distance (Z).

 

PROPERTY_DS_DISPARITY_SHIFT

The disparity shift value, in millimeters. Both the minimum depth value and the maximum depth value are shifted to enhance the resolution of depth values in between.

 

PROPERTY_DS_LEFTRIGHT_EXPOSURE

The depth stream exposure value, in log base 2 seconds.

 

PROPERTY_DS_LEFTRIGHT_GAIN

The depth stream gain adjustment, with negative values darker and positive value brighter, and zero as normal.

 

PROPERTY_DS_MIN_MAX_Z

Set the range of the depth value, in the PXC[M]RangeF32 structure.

 

PROPERTY_DS_TEMPERATURE

The temperature of the camera head in Celsius.

 

Miscellaneous Properties

 

 

PROPERTY_PROJECTION_SERIALIZABLE

The metadata identifier of the PXC[M]Projection implementation, attached to the session meta data.

 

PROPERTY_CUSTOMIZED

Device specific customized controls. Any customized controls should have a label value larger than PROPERTY_CUSTOMIZED.

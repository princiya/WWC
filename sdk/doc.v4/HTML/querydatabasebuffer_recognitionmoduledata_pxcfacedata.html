Parameters

 

buffer

The application-allocated buffer.

Description

The QueryDatabaseBuffer function copies the recognition database to the specified buffer. The application must allocate the output buffer, whose size can be obtained through the QueryDatabaseSize function.

Return Status

 

true if the function returned successfully, or false if there is any error.

Namespace Hierarchy

JavaScript

PXCMSenseManager.EnableHand

Syntax

JavaScript

function EnableHand(onData);

 

Parameters

 

onData

The callback function OnData to receive per-frame hand tracking data.

Description

The EnableHand function activates the hand module (PXC[M]HandModule) in the pipeline.

Return Status

 

The PXC[M]HandModule instance as a promise object.

Remarks

JavaScript

The EnableHand function takes a callback function OnData(mid, module, data) at input. The callback function is invoked when there is any active hand tracking data.

See Table 19 for the function parameter definitions.

See also Hand Tracking for concepts and programming guide.

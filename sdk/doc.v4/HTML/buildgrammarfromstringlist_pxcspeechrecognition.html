Parameters

 

gid

The grammar identifier. This can be any unique non-zero number.

 

cmds

The list of command words.

 

label

The optional list of labels that correspond to the commands. If not specified, the labels are 1...ncmds.

 

ncmds

The number of commands in the list.

Description

The BuildGrammarFromStringList function builds a command & control grammar with the specified list of commands. Each command is a NULL-terminated text string of words.

Command examples:

▪A game instruction: jump up
▪A volume control command: computer volume up please.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_PARAM_UNSUPPORTED

Unable to add the command for unsupported annotations or duplicated commands.

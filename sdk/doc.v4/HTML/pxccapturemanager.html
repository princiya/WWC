PXCMCaptureManager

The PXC[M]CaptureManager interface is a helper interface (on top of the PXC[M]Capture interface) for creating complex pipelines. The general concept is that there are a set of processing modules (each with a unique identifier.) The PXC[M]CaptureManager interface provides a set of functions to help consolidate the input needs of these processing modules and locate the right I/O device that can provide data to satisfy all input needs.

The application can use the CreateCaptureManager function to create an instance of the PXC[M]CaptureManager interface.

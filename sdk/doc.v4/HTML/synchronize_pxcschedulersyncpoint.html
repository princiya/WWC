Parameters

 

timeout

The time out value in milliseconds.

Description

The Synchronize function waits until an asynchronous execution is completed, or returns due to time out.

If timeout is omitted, the function waits infinitely until an asynchronous execution is completed.

Return Status

 

PXC[M]_STATUS_EXEC_TIMEOUT

The asynchronous operation is still running. The function returns due to time out.

 

Other values

The return status of the completed asynchronous operation.

Remarks

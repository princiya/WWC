Face Tracking via SenseManager

 Top  Previous  Next

To activate face tracking algorithm in the PXC[M]SenseManager pipeline, use the EnableFace function. You can implement face tracking in your application in two ways: using the SenseManager procedural functions or using the SenseManager callback functions.

Face Tracking Using the SenseManager Procedural Functions

Example 41 shows an example of a typical face detection application. The example does the following:

▪Create an instance of the PXC[M]SenseManager interface.
▪Activate face tracking by using the EnableFace function.  At this point, you may retrieve an instance of the PXC[M]FaceModule interface (through the QueryFace function) for customized operations, such as configuring the face tracking features.

The PXC[M]FaceModule instance is valid until you calls the Close function to close the pipeline.

▪Initialize the PXC[M]SenseManager pipeline (Init) and loop through the image frames (AcquireFrame and ReleaseFrame). Whenever some face analysis result is available, such as a face is detected, the QueryFace function returns a valid instance.

Such checking with the QueryFace function is optional if you calls the AcquireFrame function with ifall=true, which blocks until all modality results are ready for processing.

▪During streaming, you may call the PauseFace function to pause/resume face tracking under certain conditions.
▪Clean up the sample.

C++ Example 41: Face Tracking using the SenseManager Procedural Functions

// Create the SenseManager instance

PXCSenseManager *sm=PXCSenseManager.CreateInstance();

 

// Enable face tracking

sm->EnableFace();

 

// Get a face instance (or inside the AcquireFrame/ReleaseFrame loop) for configuration.

PXCFaceModule *face=sm->QueryFace();

...

 

// Initialize the pipeline

sm->Init();

 

// Stream data

while (sm->AcquireFrame(true)>=PXC_STATUS_NO_ERROR) {

  // retrieve the face tracking results

  PXCFaceModule *face2=sm->QueryFace();

  if (face2) {

      ...

  }

 

  // Resume next frame processing

  sm->ReleaseFrame();

}

 

// Clean up

sm->Release();

Face Tracking Using the SenseManager Callback Functions

You can also use the PXC[M]SenseManager callback functions. Example 42 shows the face tracking application using callbacks. The SDK invokes the OnModuleProcessedFrame callback when there is some data for processing.

C++ Example 42: Face Tracking using the SenseManager Event Functions

class MyHandler: public PXCSenseManager::Handler {

public:

   virtual pxcStatus PXCAPI OnModuleProcessedFrame(pxcUID mid, PXCBase *module, PXCCapture::Sample *sample) {

       // check if the callback is from the face tracking module.

       if (mid==PXCFaceModule::CUID) {

           PXCFaceModule *face=module->QueryInstance<PXCFaceModule>();

           ...

       }

       // return NO_ERROR to continue, or any error to abort.

       return PXC_STATUS_NO_ERROR;

   }

};

 

// Create the SenseManager instance

PXCSenseManager *sm=PXCSenseManager.CreateInstance();

 

// Enable face tracking

sm->EnableFace();

 

// Get a face instance for feature configuration

PXCFaceModule *face=sm->QueryFace();

...

 

// Initialize

MyHandler handler;

sm->Init(&handler);

 

// Streaming data

sm->StreamFrames(true);

 

// Clean up

sm->Release();

Description

The RestorePropertiesUponFocus function restores the device properties to the values that the application set previously.

When multiple SDK applications running on the same system, these applications may set different device properties thus impact each other in experience. The guideline is that when an application receives the windows focus, the application should immediately call the RestorePropertiesUponFocus function to restore any device properties that the application previously set. The function keeps a record of all device properties changes internally.

See Coexisting with Other Applications for more details.

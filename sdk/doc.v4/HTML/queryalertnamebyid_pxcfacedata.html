Syntax

C++

pxcStatus  QueryAlertNameByID(AlertType alertEvent, pxcCHAR *alertName);

 

C#

pxcmStatus QueryAlertNameByID(AlertType alertEvent, out String alertName);

 

Java

pxcmStatus QueryAlertNameByID(AlertType alertEvent, StringBuilder alertName);

 

Parameters

 

alertEvent

The alert event as defined in the AlertType enumerator.

 

alertName

The name of the alert event, to be returned.

Description

The QueryAlertNameByID function returns the alert event name, if the specified event is fired.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_DATA_UNAVAILABLE

The specified alert was not fired.

Remarks

C++

The C++ application must allocate the alert name buffer, the size of which is ALERT_NAME_SIZE.

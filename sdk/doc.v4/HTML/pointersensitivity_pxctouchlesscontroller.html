Description

The PointerSensitivity enumerator itemizes the supported pointer movement sensitivity levels.

Name/Description

 

PointerSensitivity_Smoothed

Smooth movement.

 

PointerSensitivity_Balanced

Slight smoothing to balance the smoothness and sensitivity.

 

PointerSensitivity_Sensitive

Non-smoothed movement.

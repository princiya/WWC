PXCMTouchlessController.ProfileInfo.Configuration

Description

The Configuration enumerator itemizes the supported configuration options.

See Handling Gestures for the gesture definition described in this enumerator.

Name/Description

 

Configuration_None

No option is selected. Use the default setting.

 

Configuration_Allow_Zoom

Allow zooming.

 

Configuration_Use_Draw_Mode

Allow continuous touch + movement interaction. Like drawing on a screen.

 

Configuration_Scroll_Horizontally

Enable horizontal scrolling using the pinch gesture.

 

Configuration_Scroll_Vertically

Enable vertical scrolling using the pinch gesture.

 

Configuration_Meta_Context_Menu

Fire up the context menu using the V gesture.

 

Configuration_Enable_Injection

Disable the injection of keyboard/mouse events.

 

Configuration_Edge_Scroll_Horizontally

Enable horizontal scrolling when the mouse cursor is at the edge of the screen.

 

Configuration_Edge_Scroll_Vertically

Enable vertical scrolling when the mouse cursor is at the edge of the screen.

 

Configuration_Hide_Cursor_After_Touch_Injection

Hide the mouse cursor after touch injection.

 

Configuration_Allow_Back

Enable the back gesture.

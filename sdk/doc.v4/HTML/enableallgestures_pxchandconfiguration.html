Parameters

 

continous

If true, enable event notification continuously during gesture formation. If false, enable event notification only at the start and stop point of the gesture recognition. If omitted, the default is false.

Description

The EnableAllGestures function enables all gesture event notifications.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

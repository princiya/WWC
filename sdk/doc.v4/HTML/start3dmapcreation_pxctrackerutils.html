Parameters

 

size

Specify the target object size. See the ObjectSize enumerator for definitions.

Description

The Start3DMapCreation function starts the 3D map creation.

The map creation process stops when you call the Cancel3DMapCreation function or the PauseTracker(true) function.

You can then save the 3D map with the Save3DMap function.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

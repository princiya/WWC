Description

The QueryEmotionSize function returns the number of detected emotions for a given frame.

The returned value is a fixed value. The application must compare the scores of all detected emotion states to determine which is the most likely. See the section Intensity and Evidence for details.

Return Status

The number of detected emotions in a frame.

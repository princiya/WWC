Parameters

 

action

The Action enumerator to describe the fired action.

Description

The touchless controller invokes this OnFiredAction function when a registered action mapping is triggered. See the AddGestureActionMapping function for registering action mapping.

The application should not implement any lengthy processing that blocks execution in this function.

Description

The ProfileInfo structure describes the voice command module configurations.

Members

 

speaker

Optional speaker name. If provided, the SDK module will adapt speech recognition to the speaker.

The C# application must use get/set functions to access this field.

 

language

The language identifier. See the LanguageType enumerator for definitions.

 

alerts

Supported alert messages. If the application specifies a mask, the SDK module will only send the specified alert messages to the alert handler function. See the AlertType enumerator for definitions.

 

endOfSentence

Suggested length (in milliseconds) for end of sentence detection.

 

threshold

The recognition confidence threshold, ranging from 0 to 100, at which the algorithm should trigger a recognition notification. See the section About Confidence Level for additional details.

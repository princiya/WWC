Description

The PXC[M]TouchlessController interface provides member functions to control the touchless controller feature.

The touchless controller feature works in the PXC[M]SenseManager pipeline. The application can retrieve an instance of the PXC[M]TouchlessController interface through the QueryTouchlessController function, after activating the feature (via the EnableTouchlessController function).

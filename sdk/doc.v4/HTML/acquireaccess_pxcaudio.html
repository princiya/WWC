Parameters

 

access

The access type; see the Access enumerator for details.

 

format

The optional audio sample format. See the AudioFormat enumerator for details.

 

options

The audio options. See the Option enumerator for details.

 

data

The output AudioData structure, to be filled.

Description

The AcquireAccess function locks the audio buffer for exclusive access.

If the specified audio format is not available in the audio storage, the function creates an internal copy that matches the audio format.

If the audio format parameter is omitted, the function uses the audio format when the audio buffer was created.

The function returns the internal data buffer in the output AudioData structure.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_DEVICE_BUSY

Failed to lock the audio buffer for exclusive access.

 

PXC[M]_STATUS_PARAM_INVALID

The specified accelerator and format combination are not supported.

Example

Example 126 shows how to access an audio object.

C++ Example 126: Access Data  of an Audio Object

PXCAudio::AudioData data;

// Request as PCM

audio->AcquireAccess(PXCAudio::READ_ACCESS, PXCAudio::AUDIO_FORMAT_PCM, &data);

...     // audio buffer is in data.buffer with the number of samples in data.dataSize

audio->ReleaseAccess(&data);

C# Example 126: Access Data of an Audio Object

PXCMAudio.AudioData data;

// Request as PCM

audio.AcquireAccess(PXCMAudio.Access.READ_ACCESS, PXCMAudio.AudioFormat.AUDIO_FORMAT_PCM, out data);

...   // audio buffer is in unmanaged memory pointed to by data.buffer (IntPtr)

...   // use data.ToByteArray() to copy the audio data to a managed byte array.

audio.ReleaseAccess(data);

Java Example 125: Access Data of an Audio Object

PXCMAudio.AudioData data=new PXCMAudio.AudioData();

// Request as PCM

audio.AcquireAccess(PXCMAudio.Access.READ_ACCESS, PXCMAudio.AudioFormat.AUDIO_FORMAT_PCM, data);

...   // use data.ToByteArray() to copy the audio data.

audio.ReleaseAccess(data);

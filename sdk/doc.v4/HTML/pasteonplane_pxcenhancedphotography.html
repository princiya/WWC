bottomLeft

The bottom left corner that the image should paste to.

Description

The PasteOnPlane function pastes the specified image into the photo image at the specified location. The function renders the specified image according to the auto-detected plane orientation of the specified location in the photo image, and the the alpha channel of the photo image.

The function creates a new photo instance at output. You must release the photo instance after use.

Return Status

 

The resulting photo image instance, or NULL/null if there is any error.

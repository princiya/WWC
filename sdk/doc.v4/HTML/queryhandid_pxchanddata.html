The hand access order; see the AccessOrderType enumerator for definitions.

The zero-based index to enumerate all hands, or the hand identifier if accessOrder= ACCESS_ORDER_BY_ID.

The hand identifier, to be returned.

Description

The QueryHandId function enumerates the detected hands' identifiers.

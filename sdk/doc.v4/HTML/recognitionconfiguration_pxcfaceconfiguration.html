Description

The RecognitionConfiguration interface provides member functions to configure the face recognition algorithm.

The application can use the QueryRecognition function to create an instance of this interface.

Data Members

 

storageDesc

The configuration parameters for the recognition database. See the RecognitionStorageDesc structure for details.

 

storageName

The configuration parameters for the recognition database name.

 

properties

The configuration parameters for the recognition algorithm. See the RecognitionProperties structure for details.

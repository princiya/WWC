Parameters

 

interval

The interval value, in second.

Description

The SetInactivityInterval function sets the algorithm inactivity interval value. The power state is switched to the power saving mode if there is extended period of inactivity.

The inactivity definition is module specific. For example, for the hand module, inactivity is defined as there is no hand in the scene. The SDK consolidate all running module in-activities to determine the camera/device power state.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

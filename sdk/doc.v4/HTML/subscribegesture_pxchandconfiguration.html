Parameters

 

handler

The gesture event handler instance.

Description

The SubscribeGesture function registers an event handler object for the gestures.

The application must maintain the life cycle of the event handler until after calling the UnsubscribeGesture function.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

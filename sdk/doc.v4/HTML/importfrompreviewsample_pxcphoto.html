sample

A preview sample instance from SenseManager streaming functions.

Description

The ImportFromPreviewSample function imports the photo content from any preview samples. The preview sample must contain the following elements:

•Color: The color image of the photo picture.
•Depth: The depth image of the photo picture.
•Projection: The projection instance to map between the color and depth images.

The PXC[M]Projection instance is attached to the depth image meta data (with identifier METADATA_DEVICE_PROJECTION). If the depth image is not aligned to the color resolution, the CreateDepthImageMappedToColor function is used to align the two images.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

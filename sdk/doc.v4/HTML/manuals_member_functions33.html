Member Functions

The RecognitionConfiguration interface exposes the following member functions:

Member Function

Description

Enable

Enable face recognition.

Disable

Disable face recognition.

SetAccuracyThreshold

Set the recognition accuracy threshold.

GetAccuracyThreshold

Get the recognition accuracy threshold.

SetRegistrationMode

Set the registration mode for adding new users to the recognition database.

GetRegistrationMode

Get the registration mode.

UseStorage

Set the current recognition database.

QueryActiveStorage

Get the current recognition database.

CreateStorage

Create a recognition database.

SetStorageDesc

Update the recognition database.

DeleteStorage

Delete the recognition database.

SetDatabaseBuffer

Import the recognition database.

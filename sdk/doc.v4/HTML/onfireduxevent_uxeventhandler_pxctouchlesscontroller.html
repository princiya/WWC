data

The UXEventData structure to describe the fired UI event parameters.

Description

The touchless controller invokes this OnFiredUXEvent function when an UI event is triggered. See the SubscribeEvent function for subscribing an UI event.

The application should not implement any lengthy processing that blocks execution in this function.

PXCSessionService::DLLExportTable

Definition

C++

struct DLLExportTable {

 DLLExportTable        *next;

 pxcStatus (PXCAPI        *createInstance)(PXCSession *session, PXCSchedulerService *scheduler, PXCAccelerator *reserved, DLLExportTable *table, pxcUID cuid, PXCBase **instance);

 pxcUID                suid;

 ImplDesc                desc;

};

Description

The DLLExportTable structure describes a module implementation and its constructor function. You can link multiple module implementations together with the next field.

Members

 

next

Point to the next implementation description table.

 

createInstance

The function pointer to create an instance of the implementation.

 

suid

Must be PXCSessionService::SUID_DLL_EXPORT_TABLE.

 

desc

The module descriptor.

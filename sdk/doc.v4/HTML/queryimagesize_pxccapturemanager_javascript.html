PXCMCaptureManager.QueryImageSize

Syntax

JavaScript

function QueryImageSize(stream);

Parameters

 

stream

The stream type number as defined in the StreamType enumerator.

Description

The QueryImageSize function returns the resolution of the specified stream. The resolution information is available only after the pipeline is initialized.

Return Status

 

The returned promise object contains:

 

sts

The status code, as defined in the pxcmStatus enumerator.

 

size

The image resolution { width, height }. See also PXCMSizeI32.

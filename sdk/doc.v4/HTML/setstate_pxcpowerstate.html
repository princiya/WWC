Parameters

 

state

The desired power state. See the State enumerator for details.

Description

The SetState function sets the module power state.

The SetState function works only when an active device is available. For the application that uses the PXC[M]SenseManager interface, the application can only use the SetState function after the Init function.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_PARAM_UNSUPPORTED

The module does not support the specified power state.

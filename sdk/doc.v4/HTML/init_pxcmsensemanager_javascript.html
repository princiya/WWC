Namespace Hierarchy

JavaScript

PXCMSenseManager.Init

Syntax

JavaScript

function Init(onConnect, onStatus);

 

Parameters

 

onConnect

The optional callback OnConnect(data) when there is a device connection/disconnection event.

 

onStatus

The optional callback OnStatus(data) when the module returns any warning or the pipeline terminates.

Description

The Init function configures the pipeline to be ready for processing.

Return Status

 

The returned promise object contains:

 

sts

The status code, as defined in the pxcmStatus enumerator.

Remarks

JavaScript

The OnConnect(data) function is invoked when there is a device connection/disconnection event. The device connection status is passed as data.connected. The function return status is ignored. See also OnConnect.

The OnStatus(data) function is invoked when there is a module processing warnings or the SenseManager pipeline terminates. The module identifier is passed as data.mid and the status code is passed as data.sts. The function return status is ignored. See also OnStatus.

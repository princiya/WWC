Description

The PXC[M]Calibration interface provides functions to retrieve the device calibration parameters.

You can use the QueryInstance function to retrieve an instance of the PXC[M]Calibration interface from the PXC[M]Projection instance.

Example

Example 128 shows how to create a calibration instance.

C++ Example 128: Create a Projection Instance

/* device is an instance of the PXCCapture::Device interface */

PXCProjection *projection=device->CreateProjection();

 

/* Get a calibration instance */

PXCCalibration *calib=projection->QueryInstance<PXCCalibration>();

...

 

/* Release the interface */

projection->Release();

C# Example 128: Create a Projection Instance

/* device is an instance of the PXCMCapture.Device interface */

PXCMProjection projection=device.CreateProjection();

 

/* Get a calibration instance */

PXCMCalibration calib=projection.QueryInstance<PXCCalibration>();

...

 

/* Dispose the interface */

projection.Dispose();

Java Example 127: Create a Projection Instance

/* device is an instance of the PXCMCapture.Device interface */

PXCMProjection projection=device.CreateProjection();

 

/* Get a calibration instance */

PXCMCalibration calib=(PXCMCalibration)projection.QueryInstance(PXCMCalibration.CUID);

...

 

/* Release the interface */

projection.close();

Parameters

 

dllName

The module shared library name.

Description

The UnloadImplFromFile function removes a module implementation. The shared library is unloaded from memory.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_ITEM_UNAVAILABLE

Failed to locate the specified module.

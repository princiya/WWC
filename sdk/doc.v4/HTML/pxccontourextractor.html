Description

The PXC[M]ContourExtractor interface provides a simple contour extractor algorithm. The input can be any color or depth image.

Example

Example 152 shows how to use the PXC[M]ContourExtractor interface.

C++ Example 152: Use the Contour Extractor

// session is a PXCSession instance

// Get the the contour extractor instance

PXCContourExtractor *ce=0;

session->CreateImpl<PXCContourExtractor>(&ce);

 

// Contour extracting operations

...

 

// Clean up

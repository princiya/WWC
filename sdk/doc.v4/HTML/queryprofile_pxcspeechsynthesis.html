Parameters

 

pidx

The zero-based configuration index.

 

pinfo

The ProfileInfo structure to retrieve the configuration parameters.

Description

The QueryProfile function returns the algorithm configuration parameters.

If pidx is omitted, the function returns the current working configuration.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_ITEM_UNAVAILABLE

The specified configuration was not found.

Description

The AcquireSegmentedImage function returns the segmented image. The application must release the image instance after use.

The resolution of the output image always matches the color image resolution. The output image format uses the color format PIXEL_FORMAT_RGB32 and contains a copy of the input color data and a synthesized alpha channel (mask). Pixels which correspond to the background will contain an alpha channel of zero, and pixels which correspond to the user will have an alpha value greater than zero.

Return Status

 

The image instance, or NULL/null if there is any error.

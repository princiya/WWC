Description

The Action enumerator itemizes the supported OS keyboard/mouse injection commands.

Name/Description

 

Action_None

No action is injected.

 

Action_LeftKeyPress

Inject a left-arrow key press action.

 

Action_RightKeyPress

Inject a right-arrow key press action.

 

Action_BackKeyPress

Inject a back-space key press action.

 

Action_PgUpKeyPress

Inject a page-up key press action.

 

Action_PgDnKeyPress

Inject a page-down key press action.

 

Action_VolumeUp

Inject an increasing volume action.

 

Action_VolumeDown

Inject a decreasing volume action.

 

Action_Mute

Inject a mute action.

 

Action_NextTrack

Inject a next track action.

 

Action_PrevTrack

Inject a previous track action.

 

Action_PlayPause

Inject a pause-playing action.

 

Action_Stop

Inject a stop-playing action.

 

Action_ToggleTabs

Inject a TAB key press action.

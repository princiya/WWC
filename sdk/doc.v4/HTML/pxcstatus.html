Description

The pxc[m]Status enumerator itemizes status codes returned by SDK functions.

Name/Description

 

Successful operation

 

PXC[M]_STATUS_NO_ERROR

The operation completed successfully.

 

Programming errors

 

PXC[M]_STATUS_HANDLE_INVALID

The session/module instance or pointer is invalid.

 

PXC[M]_STATUS_ALLOC_FAILED

Failed to allocate memory or create an instance of a module or resource.

 

Configuration-related errors or warnings

 

PXC[M]_STATUS_FEATURE_UNSUPPORTED

The requested feature is not available or not implemented.

 

PXC[M]_STATUS_PARAM_UNSUPPORTED

There are invalid/unsupported parameters in the configuration.

 

PXC[M]_STATUS_ITEM_UNAVAILABLE

The item could not be found, or end of stream.

 

PXC[M]_STATUS_INIT_FAILED

General failure during algorithm initialization.

 

Asynchronous operation-related errors or warnings

 

PXC[M]_STATUS_EXEC_ABORTED

The asynchronous pipeline operation is aborted in upstream components. Check the status return code of upstream components to determine where the error occurred.

 

PXC[M]_STATUS_EXEC_INPROGRESS

The asynchronous operation is in progress.

 

PXC[M]_STATUS_EXEC_TIMEOUT

The synchronization function timed out. The asynchronous operation is still in progress.

 

Hardware device-related errors or warnings

 

PXC[M]_STATUS_DEVICE_FAILED

Unexpected device failure due to device mulfunctioning, failed to allocate acceleration resources, etc. The application must close the module instance and recreate it.

 

PXC[M]_STATUS_DEVICE_LOST

Unexpected device failure due to the loss of the device such as system sleep/hibernation, or the peripheral device was unplugged. The application can choose to abort or wait for the device to reconnect. In the latter case, the device operation will resume.

 

PXC[M]_STATUS_DEVICE_BUSY

The application does not have control of the device thus cannot change the device configuration. The application can only use the existing configuration of the device.

 

File-based errors or warnings

 

PXC[M]_STATUS_FILE_WRITE_FAILED

Failed to open a file for write or write to a file.

 

PXC[M]_STATUS_FILE_READ_FAILED

Failed to open a file for read or read a file.

 

PXC[M]_STATUS_FILE_CLOSE_FAILED

Failed to close a file handle.

 

Data & process errors and warnings

 

PXC[M]_STATUS_DATA_UNAVAILABLE

Data not available for module processing.

 

PXC[M]_STATUS_DATA_NOT_INITIALIZED

Failed to initialize data.

 

PXC[M]_STATUS_PROCESS_FAILED

General failure in algorithm processing.

 

PXC[M]_STATUS_DATA_NOT_CHANGED

Data remains the same as no new input is available.

 

Miscellaneous errors or warnings

 

PXC[M]_STATUS_TIME_GAP

There is a gap in time stamps.

 

PXC[M]_STATUS_PARAM_INPLACE

The algorithm is already configured with the same parameters.

 

PXC[M]_STATUS_VALUE_OUT_OF_RANGE

The data value(s) is out of range.

 

PXC[M]_STATUS_COORDINATE_SYSTEM_CONFLICT

Mismatched coordinate system definitions.

Parameters

 

sample

The input stream sample required by the scene perception module. Use the QueryScenePerceptionSample function to retrieve the sample about to deliver to the scene perception module.

Description

The CheckSceneQuality function returns whether the input stream is suitable for starting scene perception or not. Use this function to determine when to start or reset scene perception, when the module is in the paused mode. You can pause/resume scene perception using the PauseScenePerception function.

Return Status

 

The quality value in between 0 (unsuitable scene to start scene perception) and 1 (ideal scene to start scene perception.)

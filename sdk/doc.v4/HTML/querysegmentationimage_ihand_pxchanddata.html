Parameters

 

image

The segmentation image, to be returned.

Description

The QuerySegmentationImage function returns the 2D image mask of the tracked hand.

The segmentation image is a 8-bit GRAY image. Each pixel is either white (0xff) representing the hand, or black (0x00) representing the background.

The image is internally managed. The application should not release the image instance. The life cycle of the image instance is the same as the PXC[M]HandData instance, or until the application calls the Update function.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

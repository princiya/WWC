Syntax

C++

pxcStatus SetFileName(pxcCHAR *file, pxcBool record);

 

C#

pxcmStatus SetFileName(String file, Boolean record);

 

Java

pxcmStatus SetFileName(string file, boolean record);

 

Parameters

 

file

The full path of the file to playback or to be recorded.

 

record

If true, set the recording mode. Otherwise, set the playback mode.

Description

The SetFileName function sets mode for file playback or recording.

For file playback, the SDK immediately creates the PXC[M]Capture instance (using the QueryCapture function), so that the application can query the capabilities of the recorded content.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

Remarks

Unity Web

File recording and playback are not supported in Unity Web. This function is not supported.

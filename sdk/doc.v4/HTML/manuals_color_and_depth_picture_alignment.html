Capturing Aligned/Unaligned Color and Depth Streams

 Top  Previous  Next

There are two ways of capturing streams for applications that work on two or more streams:

•Aligned Capturing: The application synchronizes the arrival time of the frames and processes the data at once.
•Unaligned Capturing: The application handles each sample (of some stream) individually when the sample is ready. Each stream may deliver samples in different frame rates.

Use the PXC[M]SenseManager interface to capture samples from multiple streams, either aligned or unaligned.

Aligned Capturing

Example 23 shows how to capture aligned color and depth samples. The example does the following:

•Select the color and depth streams using the EnableStream function, and then initializes the pipeline using the Init function.
•In the loop, use the AcquireFrame function with ifall=true to wait for both samples to be ready.
•Retrieve the color and depth samples using the QuerySample function.
•Release the frame for reading next sample using the ReleaseFrame function.
•Use the Close  function to clean up.

C++ Example 23: Capture Aligned Color and Depth Samples using the SenseManager

void CaptureAlignedColorDepthSamples(void) {

   // Create a SenseManager instance

   PXCSenseManager *sm=PXCSenseManager::CreateInstance();

 

   // Select the color and depth streams

   sm->EnableStream(PXCCapture::STREAM_TYPE_COLOR,640,480,30);

   sm->EnableStream(PXCCapture::STREAM_TYPE_DEPTH,320,240,30);

 

   // Initialize and Stream Samples

   sm->Init();

   for (;;) {

       // This function blocks until both samples are ready

       if (sm->AcquireFrame(true)<PXC_STATUS_NO_ERROR) break;

 

       // retrieve the color and depth samples aligned

       PXCCapture::Sample *sample=sm->QuerySample();

 

       // work on the samples sample->color and sample->depth

       ...

 

       // go fetching the next samples

       sm->ReleaseFrame();

   }

 

   // Close down

   sm->Release();

}

Unaligned Capturing

Example 24 shows how to capture unaligned color and depth samples. The example does the following:

•Select the color and depth streams using the EnableStream function, and then initializes the pipeline using the Init function.
•In the loop, use the AcquireFrame function with ifall=false to wait for any sample that is ready to read.
•Retrieve the sample using the QuerySample function. Check the color or depth sample readiness and process accordingly.
•Release the frame for reading next sample using the ReleaseFrame function.
•Use the Close  function to clean up.

C++ Example 24: Capture Unaligned Color and Depth Samples using the SenseManager

void CaptureUnalignedColorDepthSamples(void) {

   // Create a SenseManager instance

   PXCSenseManager *sm=PXCSenseManager::CreateInstance();

 

   // Select the color and depth streams

   sm->EnableStream(PXCCapture::STREAM_TYPE_COLOR,640,480,30);

   sm->EnableStream(PXCCapture::STREAM_TYPE_DEPTH,320,240,30);

 

   // Initialize and Stream Samples

   sm->Init();

   for (;;) {

       // This function blocks until any sample is ready

       if (sm->AcquireFrame(false)<PXC_STATUS_NO_ERROR) break;

 

       // retrieve the color and depth samples unaligned

       PXCCapture::Sample *sample=sm->QuerySample();

       if (sample) {

           if (sample->color) {

               // work on the color sample

               ...

           }

           if (sample->depth) {

               // work on the depth sample

               ...

           }

       }

 

       // go fetching the next samples

       sm->ReleaseFrame();

   }

 

   // Close down

   sm->Release();

}

It is also possible to capture unaligned color and depth samples using the StreamFrames function and the PXC[M]SenseManager event callbacks. Example 25 shows how to do that. The callback function OnNewSample is invoked whenever a color or depth sample is available.

C++ Example 25: Capture Unaligned Color and Depth Samples using the SenseManager Events

class MyHandler:public PXCSenseManager::Handler {

public:

 

   virtual pxcStatus PXCAPI OnNewSample(pxcUID, PXCCapture::Sample *sample) {

       if (sample->color) {

           // work on the color sample

           ...

       }

       if (sample->depth) {

           // work on the depth sample

           ...

       }

       // return NO_ERROR to continue, or any error to exit the loop

       return PXC_STATUS_NO_ERROR;

   }

};

 

void CaptureUnalignedColorDepthSamples(void) {

   PXCSenseManager *sm=PXCSenseManager.CreateInstance();

 

   // Select the color and depth streams

   sm->EnableStream(PXCCapture::STREAM_TYPE_COLOR,640,480,30);

   sm->EnableStream(PXCCapture::STREAM_TYPE_DEPTH,320,240,30);

 

   // Initialize my handler

   MyHandler handler;

   sm->Init(&hanlder);

 

   // Streaming

   sm->StreamFrames(true);

 

   // Clean up

   sm->Release();

}

To capture aligned color and depth samples using the StreamFrames function and the PXC[M]SenseManager event callbacks, you must call the EnableStreams function with the stream descriptor (DataDesc) that describes the request for the color and depth streams, as illustrated in Example 26. The callback function OnImageSamples is invoked when both color and depth samples are available, in the order specified by the stream descriptor.

C++ Example 26: Capture Aligned Color and Depth Samples using the SenseManager Events

class MyHandler:public PXCSenseManager::Handler {

public:

 

   virtual pxcStatus PXCAPI OnNewSample(pxcUID, PXCCapture::Sample *sample) {

       // Work on sample->color and sample->depth

       ...

       // return NO_ERROR to continue, or any error to exit the loop

       return PXC_STATUS_NO_ERROR;

   }

};

 

void CaptureAlignedColorDepthSamples(void) {

   // Create a SenseManager instance

   PXCSenseManager *sm=PXCSenseManager.CreateInstance();

 

   // Select the color and depth streams

   PXCVideoModule::DataDesc ddesc={};

   ddesc.deviceInfo.streams=PXCCapture::STREAM_TYPE_COLOR|PXCCapture::STREAM_TYPE_DEPTH;

   sm->EnableStreams(&ddesc);

 

   // Initialize my handler

   MyHandler handler;

   sm->Init(&hanlder);

 

   // Stream Data

   sm->StreamFrames(true);

 

   // Clean up

   sm->Release();

}

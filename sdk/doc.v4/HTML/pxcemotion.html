Description

The PXC[M]Emotion interface provides member functions for emotion detection. The application uses the CreateImpl function with the interface identifier PXCEmotion::CUID (C++)/PXCMEmotion.CUID (C#) to create an instance of the PXC[M]Emotion interface.

Example

Example 148 shows how to perform emotion detection in a live camera streaming.

C++ Example 148: Emotion Detection with Live Streaming

// Create a SenseManager instance

PXCSenseManager *sm=PXCSenseManager::CreateSenseManager();

 

// Enable emotion detection in the multimodal pipeline

sm->EnableEmotion();

 

// Get an emotion instance here (or inside the AcquireFrame/ReleaseFrame loop) for querying capabilities.

PXCEmotion *em=sm->QueryEmotion();

...

 

// Initialize the pipeline

sm->Init();

 

// Stream data

while (sm->AcquireFrame(true)>=PXC_STATUS_NO_ERROR) {

  /* retrieve emotion detection results */

  em->QueryNumFaces(...);

  ...

 

  sm->ReleaseFrame();

}

 

// Clean up

sm->Release();

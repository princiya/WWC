Configuring Pulse Estimation

To configure pulse estimation, use the PulseConfiguration interface, as shown in Example 53. The PulseConfiguration interface enables/disables the pulse estimation algorithm.

C++ Example 53: Configure Pulse Estimation

// cfg is a PXCFaceConfiguration instance

cfg->QueryPulse()->Enable();

C# Example 53: Configure Pulse Estimation

// cfg is a PXCMFaceConfiguration instance

cfg.QueryPulse().Enable();

Java Example 53: Configure Pulse Estimation

// cfg is a PXCMFaceConfiguration instance

cfg.QueryPulse().Enable();

Pulse Data

If you enable pulse estimation, you can use the QueryPulse function to retrieve any pulse data, described in the PulseData interface.

Example 54 illustrates the steps to retrieve the pulse data.

C++ Example 54: Retrieve the Pulse Data

// fdata is a PXCFaceData instance

pxcI32 nfaces=fdata->QueryNumberOfDetectedFaces();

for (pxcI32 i=0;i<nfaces;i++) {

   // Retrieve the data instance

   PXCFaceData::Face *face=fdata->QueryFaceByIndex(i);

   PXCFaceData::PulseData *pdata=fdata->QueryPulse();

 

   // retrieve the pulse rate

   pxcF32 rate=pdata->QueryHeartRate();

   ...

}

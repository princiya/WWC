PXCMBlobModule

Description

The PXC[M]BlobModule interface provides member functions to perform blob and contour extraction.

The application can use the EnableBlob function to activate hand tracking in the PXC[M]SenseManager pipeline.

Example 147 shows how to perform hand tracking with a live camera streaming.

C++ Example 147: Hand Tracking with Live Streaming

// Create a SenseManager instance

PXCSenseManager *sm=session->CreateSenseManager();

 

// Enable hand tracking in the pipeline

sm->EnableHand();

 

// Get a hand instance here (or inside the AcquireFrame/ReleaseFrame loop) for querying capabilities.

PXCHandModule *hand=sm->QueryHand();

...

 

// Initialize the pipeline

sm->Init();

 

// Stream data

while (sm->AcquireFrame(true)>=PXC_STATUS_NO_ERROR) {

  /* retrieve hand analysis results */

  hand->...

  ...

 

  sm->ReleaseFrame();

}

 

// Clean up

sm->Release();

PXCMSpeechSynthesis

Description

The PXC[M]SpeechSynthesis interface provides member functions to produce audio from text (text to speech).

The application can create an instance of this interface by using the CreateImpl function with the interface identifier PXCSpeechSynthesis::CUID (C++) or PXCMSpeechSynthesis.CUID (C#,Java).

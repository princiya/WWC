Configuring Tracking Options

 Top  Previous  Next

You can configure the hand module to produce different types of tracking data; for instance hand segmentation images, joint tracking data, etc. In addition, you can activate several features that modify the raw tracking data to produce enhanced data. The enhanced data is easier for your application to process and improves the user experience.

The following table summarizes the hand module tracking configuration options:

Function

Description

Recommended Configuration

SetTrackingMode

Set the tracking mode. (See Tracking Modes.)

Unless the joint information is needed, set to the extremities mode to conserve CPU and memory resources.

EnableTrackedJoints

Enable the retrieval of tracked joints information. (See Tracking Joints.)

 

Enable joint tracking if your application uses specific joint positions; otherwise disable in order to conserve memory resources.

EnableNormalizedJoints

Enable the calculation of a normalized skeleton. (See Skeleton Normalization.)

False. Enable the feature when your application requires a standard size hand, and not the precise locations of the specific hand joints.

EnableSegmentationImage

Enable calculation of the hand segmentation image. (See Segmentation Images.)

Set to false unless you need the segmentation image, e.g. for display purposes. Disabling the feature conserves memory (but not CPU, as segmentation is calculated in any case).

EnableJointSpeed

Activate calculation of the speed (in world coordinates and meters/second) of a specific joint. (See Tracking Joints.)

By default, joint speed calculation is disabled for all joints, in order to conserve CPU and memory resources. Leave it disabled unless it’s important for your application to be aware of hand speed.

EnableStabilizer

Enable the stabilizer feature. (See Stabilization.)

Set to “true” (stabilizer enabled).

SetSmoothingValue

Sets the degree of hand motion smoothing (a number between 0 and 1). (See Smoothing.)

The recommended smoothing value is 1.

SetUserName

Specify a name string for your application’s current user. (See Skeleton Calibration.)

N/A

SetTrackingBounds

Set the boundaries of the tracking area. (See Tracking Boundaries.)

The Hand Module’s default values. You may choose to set the tracking boundaries to be slightly smaller than the actual sensor’s field of view, in order to receive earlier alerts for out-of-boundaries events.

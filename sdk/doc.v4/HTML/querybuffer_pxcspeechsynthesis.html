Java

PXCMAudio QueryBuffer(int sid, int idx);

 

Parameters

 

sid

The sentence identifier.

 

idx

The zero-based index to retrieve multiple audio buffers.

Description

The QueryBuffer function returns the audio buffer of the specified synthesized sentence. If there are multiple buffers, call this function multiple times with an increasing idx parameter.

Return Status

 

The audio instance, or NULL/null if there is no more buffer.

Description

The StreamDesc structure describes the stream characteristics of a module’s I/O data requirements. Unused fields should be set to zero.

Members

 

sizeMin

If not zero, specify the minimum resolution.

 

sizeMax

If not zero, specify the maximum resolution.

 

frameRate

If not zero, specify the frame rate range.

 

options

If not zero, specify the stream options.

 

propertySet

The Handler interface provides the following callback functions/delegates:

Callback/Delegate

Description

OnNewSample

Events occur during streaming, when a captured image sample is available.

OnModuleProcessedFrame

 

Events occur during streaming, when processing module data is available.

OnModuleSetProfile

Events occur during initialization, before the module configuration is set.

OnConnect

Event occurs during streaming, when the device is disconnected or reconnected.

OnStatus

Event occurs during streaming, when a module issues any warning or error.

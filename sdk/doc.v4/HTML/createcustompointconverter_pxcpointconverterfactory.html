Description

The CreateCustomPointConverter function creates an instance of the PXC[M]PointConverter instance for generic use. You must use Set2DPoint or Set3DPoint to set the point data before retrieving any converted point data.

Return Status

 

The PXCMPointConverter instance, or NULL/null if there is any error.

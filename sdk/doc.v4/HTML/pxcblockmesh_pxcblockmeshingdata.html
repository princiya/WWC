Description

The PXCBlockMesh structure describes the block of mesh data.

Members

 

meshId

A unique identifier to identify the block of mesh data.

 

vertexStartIndex

The starting index of the vertice in the vertices buffer, returned by the QueryVertices function.

 

numVertices

The total number of vertices in the block of mesh data.

 

faceStartIndex

The starting index of the faces in the mesh face buffer, return by the QueryMeshFaces function.

 

numFaces

The total number of mesh faces in the block of mesh data.

 

min3dPoint

The minimum point for the axis aligned bounding box containing the mesh piece.

 

max3dPoint

The maximum point for the axis aligned bounding box containing the mesh piece

 

maxDistanceChange

The maximum change in distance due to accumulation since the block was last meshed.

 

avgDistanceChange

The average change in distance due to accumulation since the block was last meshed.

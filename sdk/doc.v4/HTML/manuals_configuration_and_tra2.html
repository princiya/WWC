Configuring Scene Perception

You can optionally configure the following aspects of the scene perception process, before the module works on the first frame of camera data:

•Initial Camera Pose: Use the SetInitialPose function to set the initial camera pose.
•Meshing Thresholds: Use the SetMeshingThresholds function to configure the meshing process (for the DoMeshingUpdate function).
•Voxel Resolution: Use the SetVoxelResolution function to set the voxel resolution. Generally, use the low resolution for reconstructing large-scale scenes, and the high resolution for small-scale scenes.

Starting Scene Quality

It is important to check the quality of the scene before starting Scene Perception. Use the CheckSceneQuality function to check the scene quality. If the function returns a value close to 1.0, the scene is good to start Scene Perception. Use the EnableSceneReconstruction function to enable scene perception, as shown in Example 98:

C++ Example 98: Checking Scene Quality Before Starting Scene Perception

// sm is a PXCSenseManager instance

// sp is a PXCScenePerception instance

PXCCapture::Sample *sample=sm->QueryScenePerceptionSample();

if (sample) {

   if (sp->CheckSceneQuality(sample)>=0.7f) {

       sp->EnableSceneReconstruction(true);

   }

}

C# Example 98: Checking Scene Quality Before Starting Scene Perception

// sm is a PXCMSenseManager instance

// sp is a PXCMScenePerception instance

PXCMCapture.Sample sample=sm.QueryScenePerceptionSample();

if (sample!=null) {

   if (sp.CheckSceneQuality(sample)>=0.7f) {

Tracking Data

Scene Perception generates the following tracking data:

•Camera Pose: The camera pose parameters contains a rotation matrix and a translation vector of the camera position in the scene. Use the GetCameraPose function to retrieve the camera pose parameters.
•Volume Preview: Use the QueryVolumePreview function to retrieve the preview image. You can render the preview image to visualize the scene reconstruction process. As more camera data becomes available, the scene becomes more detailed.
•Mesh Data: Use the CreatePXC[M]BlockMeshingData function to create a storage of the mesh data. The mesh data storage can be reused to retrieve mesh data from multiple frames. Use the DoMeshingUpdate function to generate the scene mesh data. This is a lengthy process thus you need to call the function in a dedicated thread. The mesh data includes vertices (including vertex color buffers), mesh faces organized into mesh blocks. See the PXC[M]BlockMeshingData interface for member functions to access the block mesh data. To export the mesh data to an obj file, use the SaveMesh function.

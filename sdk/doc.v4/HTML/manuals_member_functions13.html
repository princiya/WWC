Member Functions

The LandmarksData interface exposes the following member functions:

Member Function

Description

QueryNumPoints

Get the number of tracked landmark points.

QueryPoints

Get the landmark data in an array.

QueryPoint

Get the data of an individual landmark point.

QueryNumPointsByGroup

Get the number of tracked landmark points in the specified group.

QueryPointsByGroup

Get the landmark data array by the landmark group.

QueryPointIndex

Map the landmark enumerated label to the landmark index.

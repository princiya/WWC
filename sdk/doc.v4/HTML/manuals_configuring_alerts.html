Configuring Alerts

 Top  Previous  Next

The hand module can produce several types of alerts, related to the hand tracking status. This includes events such as a hand coming into view, being successfully tracked, going out of range, and so on. You can call the following functions to configure alerts:

•EnableAlert – enable alert messaging for a specific event.
•EnableAllAlerts – enable alert messaging for all events.
•SubscribeAlert – register an alert handler you have implemented in order to handle alert events, using a callback mechanism.

See Handling Alerts for more information about events that produce alerts.

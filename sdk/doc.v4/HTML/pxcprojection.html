PXCMProjection

Description

The PXC[M]Projection interface provides functions to map or project among color, depth and world coordinates.

The application can use the CreateProjection function to create an instance of the PXC[M]Projection interface.

The PXC[M]Projection interface implements the PXC[M]Calibration interface. You can get device calibration and transformation parameters from the PXC[M]Calibration interface.

C++ Example 136: Create a Projection Instance

/* device is an instance of the PXCCapture::Device interface */

PXCProjection *projection=device->CreateProjection();

 

...

 

/* Release the interface */

projection->Release();

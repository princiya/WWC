Accessing Hand Posture and Side

 Top  Previous  Next

The IHand object contains the following methods for retrieving hand posture and side data.

Side and posture information is available only in full-hand tracking mode.

•QueryBodySide – returns the side of the body (right or left) to which the hand belongs, if known.
•QueryPalmOrientation – returns a quaternion representing the global 3D orientation of the palm.
•QueryPalmRadiusImage – returns the approximate radius of the hand palm image in pixels.
•QueryPalmRadiusWorld – returns the approximate radius of the real world hand palm in meters.
•QueryOpenness – returns the degree of openness of the hand. The possible degree values range from 0 (all fingers completely folded) to 100 (all fingers fully spread).

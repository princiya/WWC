Parameters

 

pause

If true, the pipeline will stop delivering samples to the module; if false, the pipeline resumes delivering samples to the module.

Description

The PauseHand function pauses or resumes the hand tracking module (PXC[M]HandModule). If paused, the pipeline no longer delivers any input samples to the module.

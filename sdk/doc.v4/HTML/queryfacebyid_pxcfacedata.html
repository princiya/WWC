Parameters

 

faceId

The face identifier

Description

The QueryFaceByID function returns the tracked face data of the specified face identifier.

The face data instance is internally managed. The application should not release the returned instance.

Return Status

 

The face data instance, or NULL/null if the specified face is not available.

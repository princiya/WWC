The Pointer Converter Utility

 Top  Previous  Next

The PXC[M]PointConverter utility converts a point in 2D or 3D space to a point with the same number of dimensions but different value ranges. For example, you might want to convert a 2D point derived from a sensor image, whose width and height are (640, 480), to a point on your screen, whose resolution is (1280, 960). Another example is to convert the 3D world position of a fingertip, into the coordinate system of a 3D game engine. In effect, this allows you to move, stretch or shrink a coordinate system so that it corresponds to another coordinate system.

Use the following two interfaces for point conversion:

•PXC[M]PointConverterFactory: used to create the specific point converter instance
•PXC[M]PointConverter: represent a single converter instance and is used to setup and perform the conversion per frame.

You can create a point converter for specifically for the hand module's extremity points, the hand module's joint positions, the blob module's extremity points, or a generic point converter. The first three are tailored to the relative modules’ data and performs the conversion automatically once the tracking data is available. The last one is a general utility, which can be used for any input data points.

Setting-Up a Point Converter

There are 3 steps for setting up a point converter instance:

1.Create the PXC[M]PointConverterFactory instance
2.Create the specific PXC[M]Pointconverter instance
3.Set the source and target details

In step 2, you use the PXC[M]PointConverterFactory interface functions to create the point converter instances:

•CreateHandJointConverter(handData, accessOrder, index, jointType)

Creates a point converter based on the specified hand joint position with the following parameters:

▪handData: specify the hand module data instance. You need to update the output data before querying the converted point’s data.
▪accessOrder: specify how the hand should be enumerated.
▪index: specify the hand to be retrieved, in the accessOrder enumeration order.
▪jointType: defines the hand joint to be converted.

See QueryHandData for more details how to define the accessOrder, index, and JointType values.

•CreateHandExtremityConverter (handData, accessOrder, index, extremityType)

Creates a point converter based on the specified hand extremity point with the following parameters:

▪handData: specify the hand module data instance. You need to update the output data before querying the converted point’s data.
▪accessOrder: specify how the hand should be enumerated.
▪index: specify the hand to be retrieved, in the accessOrder enumeration order.
▪extremityType: defines the hand extremity point to be converted.

See QueryHandData for more details how to define the accessOrder, index, and extremityType values.

•CreateBlobPointConverter(blobData, accessOrder, index, extremityType)

Creates a point converter based on the specified blob extremity point with the following parameters:

▪blobData: specify the blob module data instance. You need to update the output data before querying the converted point’s data.
▪accessOrder: specify how the blob should be enumerated.
▪index: specify the blob to be retrieved, in the accessOrder enumeration order.
▪extremityType: defines the blob extremity point to be converted.

See QueryBlobByAccessOrder for more details how to define the accessOrder, index, and extremityType values.

•CreateCustomPointConverter()

Creates a general purpose point converter. Set the point data using Set2DPoint or Set3DPoint before retrieving the converted data.

Example 105 shows how to initialize the PXC[M]PointConverter instance for the output of the hand module.

C++ Example 105: Initialize a Point Converter for Hand

// Hand Module Setup

PXCSenseManager* senseManager = PXCSenseManager::CreateInstance();

senseManager->EnableHand();

PXCHandModule* handModule = senseManager->QueryHand();

PXCHandData* handDataOutput = handModule->CreateOutput();

 

// Create an instance of PXCPointConverterFactory

PXCPointConverterFactory* pointConverter = NULL;

senseManager->QuerySession()->CreateImpl<PXCPointConverterFactory>(&pointConverter);

 

// Create a point converter for the wrist of the first detected hand

PXCPointConverter* converter=pointConverter->CreateHandJointConverter(handDataOutput, PXCHandData::ACCESS_ORDER_BY_TIME, 0, PXCHandData::JOINT_WRIST);

Converting 2D Points

To convert 2D points, define the source and target areas that define the conversion. The areas are the rectangles. The input point is linearly projected from the source rectangle to the relative location in the target rectangle.

you don’t have to map the entire source image area. Use a smaller rectangle to allow large motions on the screen.

Example 106 maps the coordinate of the depth image to screen coordinates (1280x960).

C++ Example 106: Convert 2D Points

// Setting source image-rectangle to the entire depth image

PXCRectF32 sourceRect2D = {0,0,640,480};

converter->Set2DSourceRectangle(sourceRect2D);

 

// Setting Target to full screen size (for resolution 1280x960)

PXCRectF32 targetRect2D = {0,0,1280,960};

converter->Set2DTargetRectangle(targetRect2D);

C# Example 106: Convert 2D Points

// Setting source image-rectangle to the entire depth image

PXCMRectF32 sourceRect2D = new PXCMRectF32(0,0,640f,480f);

converter.Set2DSourceRectangle(sourceRect2D);

 

// Setting Target to full screen size (for resolution 1280x960)

PXCMRectF32 targetRect2D = new PXCMRectF32(0,0,1280f,960f);

converter.Set2DTargetRectangle(targetRect2D);

Java Example 106: Convert 2D Points

To convert 3D points, define the source and target volumes that define the conversion. The volumes are rectangular boxes. The input point is linearly projected from the source box to the relative location in the target box.

You can set the source (and target) ranges dynamically, which can be used to adapt the current working area to the user’s preferred locations (e.g. the mean hand position in the last few frames).

Example 107 sets a 3D World Box to be mapped to a user defined target box with range [0-1]. The source box is located at a natural position of the user’s right hand: 15 cm to the right of the camera-center, 5 cm below the camera-center, at 30 cm distance from the camera. The box’s size is 20 cm to each direction.

C++ Example 107: Convert 3D Points

// Setting source "World box"

PXCBox3DF32 sourceBox3D = {15,-5,30,20,20,20};

converter->Set3DSourceBox(sourceBox3D);

 

// Setting Target

PXCBox3DF32 targetBox3D = {0,0,0,1,1,1};

converter->Set3DTargetBox(targetBox3D);

C# Example 107: Convert 3D Points

// Setting source "World box"

PXCMBox3DF32 sourceBox3D = new PXCMBox3DF32();

sourceBox3D.centerOffset=new PXCMPoint3DF32(15f,-5f,30f);

sourceBox3D.dimension=new PXCMPoint3DF32(20f,20f,20f);

converter.Set3DSourceBox(sourceBox3D);

 

// Setting Target

PXCMBox3DF32 targetBox3D = new PXCMBox3DF32();

targetBox3D.centerOffset=new PXCMPoint3DF32(0,0,0);

targetBox3D.dimension=new PXCMPoint3DF32(1f,1f,1f);

converter.Set3DTargetBox(targetBox3D);

Module Based Point Converters

To use the point converters on a module’s output, you need to ensure that module’s output is up-to-date, and then query the position of the converted point. Example 109 shows the code that should be called every frame, for converters that are based on the hand module. Similar code can be used for the blob module output.

C++ Example 109: Update Module Data Before Retrieving Converted Data

handDataOutput->Update();

Full Code Sample

Example 111 uses the PXC[M]PointConverter interface with the hand module to convert the position of the index fingertip to a 3D box in a game coordinate system.

C++ Example 111: Convert 3D Points

// Init

PXCSenseManager* senseManager = PXCSenseManager::CreateInstance();

 

// Hand Module Setup

senseManager->EnableHand();

PXCHandModule* handModule = senseManager->QueryHand();

PXCHandData* handDataOutput = handModule->CreateOutput();

 

// Creating PXCPointConverter instance

PXCPointConverterFactory* pointConverterFactory;

senseManager->QuerySession()->CreateImpl<PXCPointConverterFactory>(&pointConverterFactory);

 

// Creating hand point converter with first hand entered in FOV and JOINT_INDEX_TIP

PXCPointConverter* handPointConverter = NULL;

handPointConverter = pointConverterFactory->CreateHandPointConverter(handDataOutput, PXCHandData::ACCESS_ORDER_BY_TIME, 0, PXCHandData::JOINT_INDEX_TIP);

 

// Setting source "World box"

PXCBox3DF32 sourceBox3D = {0,0,0,60,60,60};

handPointConverter->Set3DSourceBox(sourceBox3D);

 

// Setting Target

PXCBox3DF32 targetBox3D = {0,0,0,1,1,1};

handPointConverter->Set3DTargetBox(targetBox3D);

 

// First Initializing the sense manager

if (senseManager->Init() >= PXC_STATUS_NO_ERROR) {        

   // Acquiring frames from input device

   while (senseManager->AcquireFrame(true) >= PXC_STATUS_NO_ERROR) {

       // Get current frame hand output

       handDataOutput->Update();

 

       if (handDataOutput->QueryNumberOfHands() > 0) {

           PXCPoint3DF32 converted3dPoint = handPointConverter->GetConverted3DPoint();

 

           // Do some work with converted3dPoint

           //...

       }

 

       // Finished working on current frame

       senseManager->ReleaseFrame();

   } // end while acquire frame

} // end if Init

 

//Release memory

handPointConverter->Release();

pointConverterFactory->Release();

handDataOutput->Release();

senseManager->Release();

JointData Properties

 Top  Previous  Next

The JointData structure (that you can retrieve by calling the QueryTrackedJoint or QueryNormalizedJoint function) contains the following fields:

•confidence – reserved for a future confidence score feature. (Currently a value of 100 indicates that joint data exists, and 0 indicates that it doesn’t.)
•positionWorld - the joint position in 3D world coordinates (in meters).
•positionImage- the  joint position in 2D coordinates (in pixels). Note that the Z coordinate contains the depth value at point (x,y) in millimeters.
•localRotation - a quaternion representing the local 3D rotation of the vector from the joint’s parent to the current joint. The rotation is relative to the parent joint’s orientation. See Local and Global Joint Rotation for more details.
•globalOrientation - a quaternion representing the global 3D orientation, relative to the world y axis. See Local and Global Joint Rotation for more details.
•speed – the vector of the joint’s velocity in 3 directions (X, Y and Z, in meters/second).

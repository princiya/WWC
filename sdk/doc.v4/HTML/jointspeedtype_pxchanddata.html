The JointSpeedType enumerator itemizes the joint speed types.

Name/Description

 

JOINT_SPEED_ABSOLUTE

Generate the joint speed value by the difference between the current position and the previous position.

 

JOINT_SPEED_AVERAGE

Generate the joint speed value by averaging over time.

Parameters

 

image

The reference image.

 

targetID, targetID[0]

The target identifier, to be returned.

 

width

The optional image width in mm. If omitted, the default value is zero. The algorithm does not take image resolution into consideration.

 

height

The optional image height in mm. If omitted, the default value is zero. The algorithm does not take image resolution into consideration.

 

quality

Reserved.

 

extensible

The optional flag to enable/disable the extensible (learning) mode. If omitted, the default value is false.

Description

The Set2DTrackFromImage function sets the reference image for 2D tracking.

The function returns the target identifier of the added image.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

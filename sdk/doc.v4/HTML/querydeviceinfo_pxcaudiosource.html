Parameters

 

didx

The optional zero-based index to enumerate all available devices.

 

dinfo

The device parameters, to be returned.

Description

The QueryDeviceInfo function returns the available audio devices.

If didx is omitted, the function returns the current working audio device parameters.

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

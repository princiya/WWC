Setting Scroll and Pointer Sensitivity

 Top  Previous  Next

There are two additional PXC[M]TouchlessController functions for configuring the module's behavior:

•SetScrollSensitivity – this function takes a float value, which multiplies the default scrolling speed when the pointer is at one of the 4 screen edges. A value of 1 keeps the default speed, less than 1 reduces the scrolling speed, and greater than 1 increases it. The recommended value range is 0.25 to 4.
•SetPointerSensitivity – this function takes PointerSensitivitythat determines the degree to which pointer movements are “smoothed”. Pointer movements are calculated according to the tracked hand’s motion, which may be “jittery”. “Smoothing” the motion ignores small jitters, producing a more consistent motion path. The possible values are:
oPointerSensitivity_Sensitive– pointer movement follows hand movement as closely as possible.
oPointerSensitivity_Balanced– pointer movement is “smoothed” moderately in relation to the original hand movement.
oPointerSensitivity_Smoothed– pointer movemenat is “smoothed” significantly in relation to the original hand movement.

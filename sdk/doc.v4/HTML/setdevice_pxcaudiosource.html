Parameters

 

dinfo

The audio device parameters.

Description

The SetDevice function sets the working audio device.

The PXC[M]AudioSource interface supports reading from an audio file (WAV, MP3, WMV: any system decode-able audio format). To read from an audio file, the application must specify the file name in dinfo->did and then call the SetDevice function.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

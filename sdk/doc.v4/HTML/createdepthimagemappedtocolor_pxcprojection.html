Parameters

 

depth

The depth image instance.

 

color

The color image instance.

Description

The CreateDepthImageMappedToColor function creates a depth image that is spatially aligned with, and the same size as the color image. Each input depth image pixels are mapped to the output depth image.

Return Status

 

The created depth image instance. The application must release the instance after use.

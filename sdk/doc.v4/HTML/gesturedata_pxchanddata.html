Description

The GestureData structure describes the details of a gesture parameters.

Members

 

timeStamp

The time stamp when the gesture occurred, in 100ns.

 

handId

The identifier of the relevant hand, if relevant and known.

 

frameNumber

The number of the frame in which the event occurred.

 

state

The state of the gesture tracking. See the GestureStateType enumerator for definitions.

 

name

The gesture name string.

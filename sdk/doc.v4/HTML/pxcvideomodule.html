Example

The application can query this interface from any SDK video module implementation, as illustrated in Example 143.

C++ Example 143: Retrieve Video Module Common Interface

// emotion is a PXCEmotion instance

PXCVideoModule *module=emotion->QueryInstance<PXCVideoModule>();

C# Example 143: Retrieve Video Module Common Interface

// emotion is a PXCMEmotion instance

PXCMVideoModule module=emotion.QueryInstance<PXCMVideoModule>();

Java Example 142: Retrieve Video Module Common Interface

// emotion is a PXCMEmotion instance

PXCMVideoModule module=(PXCMVideoModule)emotion.QueryInstance(PXCMVideoModule.CUID);

Interface and Function Reference: Utilities

 Top  Previous  Next

This section describes the Intel® RealSense™ SDK utility interfaces and their operations for application development.

In each interface description, only commonly used status codes are documented. The function may return additional status codes in certain cases. See the pxc[m]Status enumerator for a list of all status codes.

Naming Conventions

This manual uses [M] (where M stands for "managed") in the interface names to represent interfaces that are available in a managed environment (such as C# and Java). For example, PXC[M]Capture represents the C++ interface PXCCapture and the C# or Java interface PXCMCapture.

The manual uses additional markings after the interface or method names as follows:

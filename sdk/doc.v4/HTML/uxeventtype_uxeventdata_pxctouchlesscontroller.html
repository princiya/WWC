Description

The UXEventType enumerator itemizes the UI event types.

Name/Description

 

UXEvent_StartZoom

UXEvent_Zoom

UXEvent_EndZoom

 

Zooming UI events: start zooming, during zooming and end zooming.

 

UXEvent_StartScroll

UXEvent_Scroll

UXEvent_EndScroll

 

Scrolling UI events: start scrolling, during scrolling, and end scrolling.

 

UXEvent_StartDraw

UXEvent_Draw

UXEvent_EndDraw

 

Drawing UI events: start drawing, during drawing, adn end drawing.

 

UXEvent_CursorMove

UXEvent_CursorVisible

UXEvent_CursorInvisible

 

Mouse cursor events.

 

UXEvent_Select

 

The user selected a button.

 

UXEvent_GotoStart

 

The user triggered the Windows* 8 Start screen.

 

UXEvent_ReadyForAction

The user is ready for a zooming or scrolling operation.

 

UXEvent_StartMetaCounter

UXEvent_StopMetaCounter

 

Start/Stop meta menu counter visual.

 

UXEvent_ShowMetaMenu

UXEvent_HideMetaMenu

 

Show/hide the meta menu.

 

UXEvent_MetaPinch

 

The pinch gesture is detection while in the meta mode.

 

UXEvent_MetaOpenHand

The open hand gesture is detected while in the meta mode.

 

UXEvent_Back

The user performs a back gesture.

 

UXEvent_ScrollUp

UXEvent_ScrollDown

UXEvent_ScrollLeft

UXEvent_ScrollRight

Edge scroll was started by moving cursor to the corresponding screen edge.

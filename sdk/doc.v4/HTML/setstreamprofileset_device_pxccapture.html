Parameters

 

profiles

The configuration parameters of all streams to be streamed.

Description

The SetStreamProfileSet function configures the streams parameters. All streams must be configured. The device streaming starts after a successful configuration.

If the device supports configuration change, a second call to the function stops the previous streaming and starts the new configuration. If the device does not support configuration change, the second call to the function returns an error.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function was successful.

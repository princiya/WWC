Parameters

 

maxDistanceChangeThreshold, or thresholds[0]

If the maximum change in a block exceeds this value, the block is re-meshed.

 

avgDistanceChangefAveDiffThreshold, or thresholds[1]

If the average change in a block exceeds this value, the block is re-meshed.

Description

The QueryMeshingThresholds function returns the thresholds for the DoMeshingUpdate function. The thresholds specify the magnitude of changes that can occur in any block that would be considered significant for re-meshing.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

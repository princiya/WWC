Description

The PXC[M]Rotation interface provides an utility algorithm for performing rotation on any vector. You can use the CreateImpl function to create an instance of the PXC[M]Rotation interface.

Example

Example 154 shows how to use the PXC[M]Rotation interface.

C++ Example 154: Use the Rotation Utility

// session is a PXCSession instance

// Get the the PXCRotation instance

PXCRotation *rot=0;

session->CreateImpl<PXCRotation>(&rot);

 

// Define a rotation

rot->SetFromQuaternion(...);

 

// Perform some rotation

rot->RotateVector(...);

 

// Clean up

rot->Release();

C# Example 154: Use the Rotation Utility

// session is a PXCMSession instance

// Get the the PXCMRotation instance

PXCMRotation rot;

session.CreateImpl<PXCMRotation>(out rot);

 

// Define a rotation

// Perform some rotation

Java Example 154: Use the Rotation Utility

// session is a PXCMSession instance

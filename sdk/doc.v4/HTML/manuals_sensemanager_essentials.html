The PXC[M]SenseManager interface provides the following member functions:

Member Functions

Description

Init

Init (JavaScript)

 

Initialize the pipeline.

AcquireFrame

Wait until a new frame is available and lock it for processing.

ReleaseFrame

Release the lock on the current frame.

FlushFrame

Flush any internal cache.

Close

Close the pipeline.

StreamFrames

StreamFrames (JavaScript)

 

A helper function that performs AcquireFrame/ ReleaseFrame streaming loop in a separate thread.

IsConnected

Check the device connection status.

QueryCaptureManager

QueryCaptureManager (JavaScript)

 

Retrieve the PXC[M]CaptureManager instance.

QuerySession

Retrieve the PXC[M]Session instance.

EnableModule

Insert a processing module into the pipeline.

QueryModule

Returns the processing module instance.

PauseModule

Pause/resume a processing module in the pipeline.

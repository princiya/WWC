The PXC[M]HandConfiguration interface exposes the following member functions:

Feature

Member Function

Description

Alert Handling

DisableAlert

 

Disable an alert event.

DisableAllAlerts

DisableAllAlerts (JavaScript)

 

Disable all alert events.

EnableAlert

Enable an alert event.

EnableAllAlerts

EnableAllAlerts (JavaScript)

 

Enable all alert events.

IsAlertEnabled

Check if an alert is enabled.

SubscribeAlert

Register to get alert events.

UnsubscribeAlert

Unregister to not get alert events.

Hand Tracking

SetUserName

Set the user name to save hand calibration.

QueryUserName

Get the user name.

SetDefaultAge

Set the calibration age.

QueryDefaultAge

Get the calibration age.

EnableJointSpeed

Enable joint speed calculation.

DisableJointSpeed

Disable joint speed calculation.

EnableJointSpeed

Set the joint speed mode.

DisableJointSpeed

Disable joint speed calculation.

SetTrackingBounds

Set the tracking bounds.

QueryTrackingBounds

Get the tracking bounds.

SetTrackingMode

Set the tracking mode.

QueryTrackingMode

Get the tracking mode.

SetSmoothingValue

Set the motion smoothing value.

QuerySmoothingValue

Get the motion smoothing value.

EnableNormalizedJoints

Enable/disable the calculation of normalized joints.

IsNormalizedJointsEnabled

Check if the normalized joints are calculated.

EnableSegmentationImage

Enable/disable the calculation of the hand segmentation image.

IsSegmentationImageEnabled

Check if the hand segmentation image is calculated.

EnableTrackedJoints

Enable/disable the calculation of tracked joints.

IsTrackedJointsEnabled

Check if tracked joints are calculated.

EnableStabilizer

Enable/disable the tracking stabilizer.

IsStabilizerEnabled

Check if the tracking stabilizer is activated.

EnableAllGestures

Enable all gestures.

Return the total number of gestures.

Enumerate all gesture names.

Check if a gesture is enabled.

UnsubscribeGesture

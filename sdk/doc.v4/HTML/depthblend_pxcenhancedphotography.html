Parameters

 

photo

The photo image instance.

 

image

The 2D image instance.

 

point

The point coordinates to insert the 2D image to.

 

depth

The depth value to insert the 2D image to.

 

rotation

The pitch-yaw-roll rotation, in degrees (0-360).

 

scale

The scaling factor. No scaling if scale==1.0.

Description

The DepthBlend function rotates and scales the 2D image, and then blends the 2D image onto the photo at the specified location and the depth value.

The function creates a new photo instance at output. You must release the photo instance after use.

Return Status

 

The photo image instance, or NULL/null if there is any error.

Description

The TargetingOption enumerator defines the options for the targeting mode.

Name/Description

 

NO_TARGETING_OPTIONS

Use the default targeting options.

 

OBJECT_ON_PLANAR_SURFACE_DETECTION

Detect object on a planar surface.

 

FACE

The target is to scan a face.

 

HEAD

The target is to scan a 3D head.

 

BODY

The target is to scan a body.

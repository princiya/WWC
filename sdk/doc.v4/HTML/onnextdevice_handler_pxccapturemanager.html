device

The current device instance.

Description

The OnNextDevice function is invoked after PXC[M]CaptureManager has checked the current device, and the current device does not qualify. If the function returns any failure, PXC[M]CaptureManager repeats checking the current device again. Otherwise, PXC[M]CaptureManager continues onto checking the next device.

This function is used to check multiple configurations on the same device. The application should return PXC_STATUS_NO_ERROR after checking all configurations on the current device.

The application should not perform any lengthy operation in the callback function.

Return Status

 

PXC[M]_STATUS_NO_ERROR

Continue onto checking the next device.

 

Any error code

Repeat checking the current device again.

Remarks

The device instance is an object that PXC[M]CaptureManager internally manages. Do not release the device instance.

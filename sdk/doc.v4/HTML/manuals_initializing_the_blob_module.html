Initializing the Blob Module

 Top  Previous  Next

You can initialize the blob module through the PXC[m]SenseManager interface. Example 55 demonstrates how to do this.

C++ Example 55: Initialize the Blob Module

// Create an instance of the SenseManager

PXCSenseManager *sm = PXCSenseManager::CreateInstance();

 

// Enable the blob module

sm->EnableBlob();

 

// Get an instance of PXCBlobModule

PXCBlobModule* blobModule = sm->QueryBlob();

C# Example 55: Initialize the Blob Module

// Create an instance of the SenseManager

PXCMSenseManager sm = PXCMSenseManager.CreateInstance();

 

// Enable the blob module

sm.EnableBlob();

 

// Get an instance of PXCMBlobModule

PXCMBlobModule blobModule = sm.QueryBlob();

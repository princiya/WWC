Integrating with the Module

 Top  Previous  Next

Workflow Overview

Here are the main steps you need to implement in your code, to integrate with the touchless controller module:

•Create an instance of the PXC[M]SenseManager interface.
•Use the EnableTouchlessController function to activate the module by calling the  function.
•Initialize the pipeline by calling the Init function.
•Get an instance of PXC[M]TouchlessController by calling the QueryTouchlessController function.
•Optional: Configure the module by calling the QueryProfile and SetProfile functions (see Configuring the Touchless Controller Module for more details). If you don’t set the configuration options explicitly, the default settings are used, which enable only the Cursor Navigation and Select features.
•Implement and register a UI control event handler.
•Optional: Implement and register alert handlers.
•In the main loop, call the AcquireFrame function to activate frame processing. This causes UI control events to be fired (if any occurred during this frame). Then call the ReleaseFrame function to release module resources.
•Optionally, at certain stages in your application, you may want to pause or resume the module’s operation, by calling the PauseTouchlessController function.
•After exiting the main loop, call the Release function to release SDK resources.

Code Examples

Example 85 and Example 86 are code examples that demonstrate how to implement the main work flow.

C++ Example 85: Main Loop

// Initialize the Sense Manager

PXCSenseManager *senseManager=PXCSenseManager::CreateInstance();

 

// Enable hand analysis in the multimodal pipeline

senseManager->EnableTouchlessController();

 

// Initialize the pipeline

senseManager->Init();

 

// Get an instance of Touchless Controller

PXCTouchlessController *touchlessController = senseManager-> QueryTouchlessController();

 

// register for ux events

UXEventHandler eventHandler;

touchlessController->SubscribeEvent(&eventHandler);

 

// Configure Touchless Controller

ProfileInfo profInfo;

touchlessController->QueryProfile(&profInfo);

profInfo |= (PXCTouchlessController::ProfileInfo::Configuration_Allow_Selection |        PXCTouchlessController::ProfileInfo::Configuration_Enable_Injection);

touchlessController->SetProfile(&profInfo);

 

// main processing loop

while (senseManager->AcquireFrame(true)>=PXC_STATUS_NO_ERROR) {

   senseManager->ReleaseFrame();

}

 

// Release resources

senseManager->Release();

C# Example 85: Main Loop

// Initialize the Sense Manager

PXCMSenseManager senseManager = PXCMSesneManager.CreateInstance();

 

// Enable hand analysis in the multimodal pipeline

senseManager.EnableTouchlessController();

 

// Initialize the pipeline

senseManager.Init();

 

// Get an instance of touchless controller

PXCMTouchlessController touchlessController = senseManager.QueryTouchlessController();

 

// register for ux events

touchlessController.SubscribeEvent(OnTouchlessControllerUXEvent);

 

// Configure Touchless Controller

PXCMTouchlessController.ProfileInfo pinfo = new PXCMTouchlessController.ProfileInfo();

pinfo.config = PXCTouchlessController.ProfileInfo.Configuration.Configuration_Allow_Selection |        PXCTouchlessController.ProfileInfo.Configuration.Configuration_Enable_Injection);

touchlessController.SetProfile(pinfo);

 

// main processing loop

while (senseManager.AcquireFrame(true)>= pxcmStatus.PXCM_STATUS_NO_ERROR) {

   senseManager.ReleaseFrame();

}

 

// Release resources

senseManager.Dispose();

Parameters

 

pose

The optional camera pose array. If omitted, the default initial pose is used.

Description

The Reset function removes all reconstructed model (volume) information and re-initializes the volume when next stream sample is processed. The function also resets the camera pose to the specified one or otherwise uses default initial pose.

The function does not reset any data in any PXCBlockMeshingData instance. Thus you need to explicitly call the instance Reset function to stay in sync with the reconstruction model inside the scene perception module.

The camera pose [R | T] is stored in a row-major array as follows:

[r11 r12 r13 tx r21 r22 r23 ty r31 r32 r33 tz]

where R is the rotation matrix

R=[ r11 r12 r13 ]

 [ r21 r22 r23 ]

 [ r31 r32 r33 ]

and T is the translation vector:

T=[tx ty tz]

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

Tracking Joints

 Top  Previous  Next

When joint tracking is enabled, you can retrieve position and rotation data for each of the 22 joints. (See Accessing Joint Data to learn more about how the 22 hand joints are defined.)

In addition, you can enable calculation of the joint speed. Typically this feature is only activated for a single joint or for a few joints (e.g. fingertip or wrist joint), as only the overall hand speed is useful to the calling application.

See EnableTrackedJoints.

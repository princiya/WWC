Namespace Hierarchy

JavaScript

PXCMSenseManager.StreamFrames

Syntax

JavaScript

function StreamFrames();

Parameters

 

None

 

Description

The StreamFrames function starts streaming. Whenever there is any per-frame module processing data, the callback functions enabled in the Enable functions are invoked.

The application can call the Close function to terminate at any time and release the resources.

Return Status

 

The returned promise object contains:

 

sts

The return status code, as defined in the pxcmStatus enumerator.

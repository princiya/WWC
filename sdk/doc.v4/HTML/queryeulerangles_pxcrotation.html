Parameters

 

order

Reserved. Must be PITCH_YAW_ROLL. The Euler order to retrieve the Euler angles. See the EulerOrder enumerator for definitions. The default is PITCH_YAW_ROLL.

Description

The QueryEulerAngles function returns the rotation in the Euler angle representation.

Return Status

 

The rotation in the Euler angles representation. The values are in Radians.

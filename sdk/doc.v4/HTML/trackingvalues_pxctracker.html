Description

The TrackingValues structure describe the tracking parameters of a tracked object.

Members

 

state

The tracking state, defined in the ETrackingState enumerator.

 

translation

The translation matrix of the pose of the tracked object.

 

rotation

The rotation matrix of the pose of the tracked object.

 

quality

The tracking quality, described as a value between 0 (not tracked) to 1 (perfectly tracked).

 

timeElapsed

The time elapsed in ms since the last state transition.

 

trackingTimeMs

The time in ms used in processing (tracking) the respective frame.

 

cosID

The object identifier of the tracked object.

 

targetName

The name of the tracked object.

 

additionalValues

Reserved. Values provided by the sensor not expressed in the fields.

 

sensor

The sensor name that provides the values.

 

translationImage

The translation component of the pose projected onto the color image, in pixels.

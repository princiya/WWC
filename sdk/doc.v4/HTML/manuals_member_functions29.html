The RecognitionData interface exposes the following member functions:

Member Function

Description

RegisterUser

Register the current user to the database.

UnregisterUser

Unregister the current user to the database.

IsRegistered

Check if the current user is registered.

QueryUserID

Get the recognized user identifier.

QueryDatabaseSize

Get the recognition database size.

QueryDatabaseBuffer

Get the recognition database.

SetDatabaseBuffer

Replace the recognition database.

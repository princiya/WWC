Face Location Data

 Top  Previous  Next

If you enable face location detection (PXC[M]FaceConfiguration.detection.IsEnabled=true) during initialization, you can use the QueryDetection function to retrieve the face location data. There may be multiple faces in any image. Each face is set a different unique identifier. To enumerate all faces in the picture, use the QueryFaceByIndex function.

The face location data includes the following information:

•Bounding box: The bounding box is a rectangular region, in the color pixels, where the detected face is. The bounding box may or may not exactly match the detected face size. Uses the QueryBoundingRect function to retrieve the bounding box values.
•Average depth: The average depth value indicates how far away the detected face is to the camera. Use the QueryFaceAverageDepth function to retrieve the average depth value.

Example 45 illustrates the steps to retrieve the face location information.

C++ Example 45: Retrieve Face Location Data

// fdata is a PXCFaceData instance

pxcI32 nfaces=fdata->QueryNumberOfDetectedFaces();

for (pxcI32 i=0;i<nfaces;i++) {

   // Retrieve the face location data instance

   PXCFaceData::Face *face=fdata->QueryFaceByIndex(i);

   PXCFaceData::DetectionData *ddata=face->QueryDetection();

 

   // work on DetectionData

   PXCRectI32 rect;

   ddata->QueryBoundingRect(&rect);

   ...

}

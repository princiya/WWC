Description

The AddRef function increases the object reference counter.

Return Status

 

The reference counter before the increment.

Example

Example 124 shows how to increase the reference counter of a PXC[M]Image instance.

C++ Example 124: Increase the Reference Counter of an Image Object

// image is a PXCImage instance;

image->QueryInstance<PXCAddRef>()->AddRef();

C# Example 124: Increase the Reference Counter of an Image Object

// image is a PXCMImage instance;

image.QueryInstance<PXCMAddRef>().AddRef();

Java Example 123: Increase the Reference Counter of an Image Object

// image is a PXCMImage instance;

image.QueryInstance<PXCMAddRef>().AddRef();

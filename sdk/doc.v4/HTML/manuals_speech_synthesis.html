Speech Synthesis

To implement speech synthesis in your application, follow the steps below:

1.Locate the Module Implementation: Use the CreateImpl function to create an instance of the PXCSpeechSynthesis instance, as illustrated in Example 82.

C++ Example 82: Create a Speech Synthesis Module Instance

PXCSpeechSynthesis *tts=0;

session->CreateImpl<PXCSpeechSynthesis>(&tts);

C# Example 82: Create a Speech Synthesis Module Instance

PXCMSpeechSynthesis tts;

session.CreateImpl<PXCMSpeechSynthesis>(out tts);

Java Example 82: Create a Speech Synthesis Module Instance

PXCMSpeechSynthesis tts=new PXCMSpeechSynthesis();

session.CreateImpl(tts);
2.Initialize the Module: Get available configurations using the QueryProfile function and set the configuration using the SetProfile function. See Example 83.

C++ Example 83: Initialize the Speech Synthesis Module

PXCSpeechSynthesis::ProfileInfo pinfo;

tts->QueryProfile(0,&pinfo);

pinfo.language=PXCSpeechSynthesis::LANGUAGE_US_ENGLISH;

tts->SetProfile(&pinfo);

C# Example 83: Initialize the Speech Synthesis Module

PXCMSpeechSynthesis.ProfileInfo pinfo;

tts.QueryProfile(0,out pinfo);

pinfo.language=PXCMSpeechSynthesis.LanguageType.LANGUAGE_US_ENGLISH;

tts.SetProfile(pinfo);

Java Example 83: Initialize the Speech Synthesis Module

PXCMSpeechSynthesis.ProfileInfo pinfo=new PXCMSpeechSynthesis.ProfileInfo;

tts.QueryProfile(0, pinfo);

pinfo.language=PXCMSpeechSynthesis.LanguageType.LANGUAGE_US_ENGLISH;

tts.SetProfile(pinfo);

Always set the language. If not set, the default language is undetermined depending on what is currently installed on the platform.

3.Synthesize speech: Use the BuildSentence function to synthesize a sentence. The synthesized speech is stored as multiple audio buffers identified by the sentence identifier, which can be any non-zero unique value. Use the QueryBuffer function to retrieve the synthesized speech buffers, illustrated in Example 84.

You may synthesize multiple sentences before retrieving any synthesized data. Use the ReleaseSentence function to release any resources allocated for the synthesized speech.

C++ Example 84: Synthesize a Sentence Into Speech

// Synthesize the text string

tts->BuildSentence(1, L”Speak this”);

 

// Retrieve the synthesized speech

int nbuffers=tts->QueryBufferNum(1);

for (int i=0;i<nbuffers;i++) {

   PXCAudio *audio=tts->QueryBuffer(1, i);

   // send audio to the audio output device

   ...

}

 

// Clean up

tts->ReleaseSentence(1);

C# Example 84: Synthesize a Sentence Into Speech

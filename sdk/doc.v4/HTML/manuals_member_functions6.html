The IHand interface exposes the following member functions:

Member Function

Description

QueryUniqueId

Return the hand identifier.

QueryUserId (Reserved)

Return the user identifier.

QueryTimeStamp

Return the time stamp.

IsCalibrated

Check if the hand is calibrated.

QueryBodySide

Return the body side.

QueryBoundingBoxImage

Return the 2D bounding box.

QueryMassCenterImage

Return the center of mass in the 2D image coordinates.

QueryMassCenterWorld

Return the center of mass in the 3D world coordinates.

QueryPalmOrientation

Return the palm orientation.

QueryPalmRadiusImage

Return the palm radius in the image space.

QueryPalmRadiusWorld

Return the palm radius in the world space.

QueryExtremityPoint

Return the extremity point data.

QueryFingerData

Return the finger data.

QueryTrackedJoint

Return the tracked joint data.

QueryTrackingStatus

Return the tracking status.

QueryOpenness

Return the hand openness status.

QueryNormalizedJoint

Return the normalized joint data.

QuerySegmentationImage

Return the segmentation image.

HasTrackedJoints

Check if the tracked joint is available.

HasNormalizedJoints

Check if the normalized joint is available.

HasSegmentationImage

Check if the segmentation image is available.

Emotion Detection [Preview]

 Top  Previous  Next

The SDK emotion detection module estimates facial emotion states from image sequences. If there are multiple faces in an image, the module estimates the emotion states for each face.

The module supports multiple emotion states such as anger and joy, as well as general sentiment such as positive and negative. For each emotion state, the algorithm provides two scores: intensity and evidence. The application evaluates the two scores of all supported states to determine the most likely emotion state. See the Intensity and Evidence section for details.

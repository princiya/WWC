Namespace Hierarchy

JavaScript

PXCMSenseManager.EnableFace

Syntax

JavaScript

function EnableFace(onData);

 

Parameters

 

onData

The callback function OnData to receive per-frame face tracking data.

Description

The EnableFace function activates the face module (PXC[M]FaceModule) in the pipeline.

Return Status

 

The PXC[M]FaceModule instance as a promise object.

Remarks

JavaScript

The EnableFace function takes a callback function OnData(mid, module, data) at input. The callback function is invoked when there is any active face tracking data.

See Table 18 for the function parameter definitions.

See also section Face Tracking and Recognition for concepts and programming guide.

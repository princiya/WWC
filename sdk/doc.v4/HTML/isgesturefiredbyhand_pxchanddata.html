Parameters

 

gestureName

The gesture string.

 

data

The gesture parameters, as described in the GestureData structure.

Description

The IsGestureFiredByHand function returns whether the specified gesture is fired by the specified hand (identifier) and if so, returns the gesture parameters.

Return Status

 

true

The specified gesture is fired.

 

false

The specified gesture is not fired.

Description

The TrackingStatusType enumerator itemizes the supported tracking status. The value is bit-OR'ed of all available tracking statuses.

Name/Description

 

TRACKING_STATUS_GOOD

The hand is tracked.

 

TRACKING_STATUS_OUT_OF_FOV

The hand is outside the field of view (X/Y axises).

 

TRACKING_STATUS_OUT_OF_RANGE

The hand is outside the depth range.

 

TRACKING_STATUS_HIGH_SPEED

The hand is moving too fast.

 

TRACKING_STATUS_POINTING_FINGERS

The hand finger points to the camera.

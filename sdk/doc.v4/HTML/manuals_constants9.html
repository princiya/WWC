The PXC[M]HandData interface exposes the following constants:

Constant

Description

CUID

The interface identifier

NUMBER_OF_FINGERS

The maximum number of fingers of a hand.

NUMBER_OF_EXTREMITIES

The maximum number of extremities supported by the algorithm.

NUMBER_OF_JOINTS

The maximum number of joints supported by the algorithm.

RESERVED_NUMBER_OF_JOINTS

The served size of buffers to store the joint data.

MAX_NAME_SIZE

The maximum number of characters for the name of a gesture or a user.

MAX_PATH_NAME

The maximum number of characters in a file name or path.

NUMBER_OF_ALERTS

The maximum number of alerts supported by the algorithm.

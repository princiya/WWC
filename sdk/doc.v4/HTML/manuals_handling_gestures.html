Handling Gestures

 Top  Previous  Next

Gestures are specific hand motions or postures. It is often convenient to have your application respond only to pre-defined gestures, rather than having to process raw position data.

Table 11 summarizes the gestures supported in the default gesture pack. Certain pairs of gestures are similar, and gesture detection can confuse them. Therefore, it’s advisable not to enable both gestures at once. Table 11 also summarizes the restrictions. In general, it is best practice to enable only those gestures that are relevant to the specific stage in your application, rather than enabling all gestures at all times. At the beginning of each stage, enable the required gestures and disable the gestures that are not required.

You can either poll for gestures in your main processing loop, or register gesture handlers whose callback functions are called when gestures are fired.

Polling for Gestures within the Main Processing Loop

There are different ways to query for fired gestures within the main processing loop:

•Call IsGestureFired for a specific gesture.
•Call IsGestureFiredByHand for a specific gesture and a specific hand.
•Call QueryFiredGesturesNumber, and then iterate over the fired gestures by calling QueryFiredGestureData with indexes from 0 to QueryFiredGesturesNumber()–1.

Example 68 illustrates how to poll for specific gestures within the main processing loop.

C++ Example 68: Poll for Specific Gestures

// handData is the PXCHandData instance

 

// poll for gestures

PXCHandData::GestureData data;

if (handData->IsGestureFired(L“tap”, &data)) {

   // handle tap gesture

   ...

}

 

if (handData->IsGestureFired(L“wave”, &data)) {

   // handle wave gesture

   ...

}

Creating and Registering a Gesture Handler

Example 69 illustrates how to create and register a gesture handler.

C++ Example 69: Subscribe to Gestures

class MyHandler:public PXCHandConfiguration::GestureHandler {

public:

 

   virtual void PXCAPI OnFiredGesture(const PXCHandData::GestureData &data) {

       // handle gestures

       if (!wcscmp(data.name, L“tap”) {

           // handle the tap gesture

           ...

       } else if (!wcscmp(data.name, L“wave”) {

           // handle the wave gesture

           ...

       }

   }

};

 

// handModule is a PXCHandModule instance

// Subscribe to the gesture notification event

PXCHandConfiguration *handConfig=handModule->CreateActiveConfiguration();

MyHandler handler;

handConfig->SubscribeGesture(&handler);

handConfig->ApplyChanges();

 

// streaming & receiving gestures

...

 

// Clean up

handConfig->UnsubscribeGesture(&handler);

handConfig->Release();

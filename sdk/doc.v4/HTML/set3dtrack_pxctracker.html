Parameters

The XML or SLAM configuration file.

 

firstCosID, cosID[0]

The first object identifier, to be returned.

 

lastCosID, cosID[1]

The last object identifier, to be returned.

 

extensible

The optional flag to enable/disable the extensible (learning) mode. If omitted, the default value is false.

Description

The Set3DTrack function sets the reference image for 3D tracking. The configuration file can be created from the Metaio Toolbox and has a suffix of .slam or .xml.

The function returns the first and last object identifiers of the added reference objects, if there are more than one reference.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

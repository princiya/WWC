pxcmStatus SetPause(boolean pause);

 

Parameters

 

pause

If true, pause playback or recording. Otherwise, resume playback or recording.

Description

The SetPause function pauses or resumes playback or recording.

If playback is paused, the application will get the same frame at the current playback position.

If recording is paused, no frame will be written to the disk.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

Remarks

Unity Web

File recording and playback are not supported in Unity Web. This function is not supported.

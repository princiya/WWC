PXCMBase.Invoke

Syntax

 

function Invoke(name, params);

Parameters

 

name

The full function name string to invoke. For example, to invoke PXCMSesneManager.AcquireFrame, use "PXCMSenseManager_AcquireFrame".

 

params

The function input parameters. For example, use "{ifall: true, timeout: 0}" as the input parameters to PXCMSesneManager.AcquireFrame.

Description

The Invoke function is a generic mechanism (as opposite to specific JavaScript functions) to access to any exposed SDK interface functions. The function takes the function name and input parameters as input, and returns the function output.

This feature is reserved. Not all SDK functions can be accessed through Invoke. Some use aggregation to reduce overhead thus their calling protocol is different from those listed in this manual. Always use the specific JavaScript functions.

Return Status

 

The function output parameters as a promise object.

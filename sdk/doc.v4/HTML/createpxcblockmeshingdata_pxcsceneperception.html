Parameters

 

maxBlockMesh

The optional maximum number of mesh blocks in the mesh data storage.

 

maxVertices

The optional maximum number of vertices in the mesh data storage.

 

maxFaces

The optional maximum number of faces in the mesh data storage.

 

bUseColor

A optional boolean value to indicate whether color buffer is required for vertcies. If omitted, the default is true.

Description

The CreatePXC[M]BlockMeshingData function allocates the mesh data storage that can be later used to store the updated block meshing data. Use the DoMeshingUpdate function to save the mesh data.

You must release the returned instance after use.

Return Status

 

The PXC[M]BlockMeshing instance, or NULL/null if there is any error.

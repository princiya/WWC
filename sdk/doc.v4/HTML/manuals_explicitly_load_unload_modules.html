Load/Unload Modules

 Top  Previous  Next

You can explicitly load a module by calling the LoadImplFromFile function, as illustrated by Example 118. Explicitly loaded modules have precedence during the module enumerations.

C++ Example 118: Explicitly Load an SDK Module

// The application explicitly load a module

// session is a PXCSession instance

session->LoadImplFromFile(L"my_module.dll");

...

session->UnloadImplFromFile(L"my_module.dll");

C# Example 118: Explicitly Load an SDK Module

// The application explicitly load a module

// session is a PXCMSession instance

session.LoadImplFromFile("my_module.dll");

...

session.UnloadImplFromFile("my_module.dll");

Java Example 118: Explicitly Load an SDK Module

// The application explicitly load a module

// session is a PXCMSession instance

session.LoadImplFromFile("my_module.dll");

...

session.UnloadImplFromFile("my_module.dll");

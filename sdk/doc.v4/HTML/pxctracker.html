Description

The PXC[M]Tracker interface provides member functions for 2D and 3D object tracking.

You can use the EnableTracker function to activate object tracking in the PXC[M]SenseManager pipeline.

Example

Example 150 shows how to perform object tracking in a live camera streaming.

C++ Example 150: Object Tracking with Live Streaming

// Create a SenseManager instance

PXCSenseManager *sm=PXCSenseManager::CreateSenseManager();

 

// Enable object tracking

sm->EnableTracker();

 

// Get a tracker instance here for configuration.

PXCTracker *tracker=sm->QueryTracker();

...

 

// Initialize the pipeline

sm->Init();

 

// Stream data

while (sm->AcquireFrame(true)>=PXC_STATUS_NO_ERROR) {

  // Retrieve the tracking results

  ...

 

  // Resume processing next frame

  sm->ReleaseFrame();

}

 

// Clean up

sm->Release();

Description

The ObjectSize enumerator describes the relative size of the targeted object.

Name/Description

 

VERY_SMALL

The object is similar in size as a cup.

 

SMALL

The object is similar in size of a desk.

 

MEDIUM

The object is similar in size of a room.

 

LARGE

The object is similar in size of a building.

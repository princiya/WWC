Parameters

 

photo

The photo image instance.

 

point

The point coordinates in the color image.

Description

The ComputeMaskFromCoordinate function creates a mask image corresponding to the point in the color image.

The mask image is the same size of the color image. You must release the image instance after use.

Return Status

 

The PXC[M]Image instance, or NULL/null if there is any error.

Description

The FingerData structure describes the parameters of a tracked finger.

Members

 

foldedness

The degree of foldedness from 0 (folded) to 100 (extended).

 

radius

The pixel radius of the finger tip. The value is a rough indicator of the volume of the finger tip mass.

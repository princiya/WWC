Parameters

 

fidx

The zero-based face index.

 

eid

The emotion state; See the Emotion enumerator for definitions.

 

data

The parameters of the given emotion, in the EmotionData structure.

Description

The QueryEmotionData function returns the detected emotion parameters.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

 

PXC[M]_STATUS_ITEM_UNAVAILABLE

The requested face or emotion is not available.

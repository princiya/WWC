Parameters

The module identifier; usually this is the interface identifier.

 

module

The module instance.

 

sample

The audio sample that the module just worked on.

Description

The OnModuleProcessFrame function is invoked when a module has completed its processing of the current frame. The module output is ready at this point. The application can use this function to retrieve the module output.

The application should not perform any lengthy operation in the callback function.

Return Status

 

PXC[M]_STATUS_NO_ERROR

Continue the processing loop.

 

Any error code

Abort the processing loop.

Remarks

The module instance is an object that PXC[M]SenseManager internally manages. Do not release the module instance.

Description

The StreamType enumerator itemizes supported video stream types. The value can be Bit-OR'ed to represent multiple stream combinations.

Name/Description

 

STREAM_TYPE_ANY

Unspecified, unknown or any stream.

 

STREAM_TYPE_COLOR

The color stream.

 

STREAM_TYPE_DEPTH

The depth stream.

 

STREAM_TYPE_IR

The inferred stream.

 

STREAM_TYPE_LEFT

The left imager stream of a stereotypical camera.

 

STREAM_TYPE_RIGHT

The right imager stream of a stereotypical camera.

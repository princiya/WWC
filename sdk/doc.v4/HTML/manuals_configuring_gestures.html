Configuring Gestures

 Top  Previous  Next

The hand module can identify several types of hand gestures. Gestures are specific hand motions or postures, which can be used to control your application. For instance, a tap can indicate a selection, a swipe can indicate a scrolling action, and so on.

Gesture identification requires pre-trained modules called “gesture packs” to be loaded into the hand module upon initialization. The hand module automatically loads the default gesture pack during initialization.

See Handling Gestures for more information about the gestures included in the default gesture pack.

You can call the following functions in order to configure gestures:

•EnableGesture – enable identification of a specific gesture. You can choose to be alerted for every frame that contains part of the gesture, or only for the two frames in which the gesture starts and ends.
•EnableAllGestures – enable identification of all gestures.
•SubscribeGesture – register an event handler you have implemented in order to handle gesture events, using a callback mechanism.
•LoadGesturePack – reserved for support of the future additional gesture packs.

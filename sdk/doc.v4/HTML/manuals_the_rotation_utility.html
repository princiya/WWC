The Rotation Utility

 Top  Previous  Next

The PXC[M]Rotation class is a utility that allows conversion between different rotation representations of points in 3D space. It supports several commonly-used rotation algorithms. Quaternion, rotation matrix, Euler angles and Angle-Axis are all ways of representing 3D rotations.  As of today, there is no unified method to handle rotation, and each algorithm uses its own conventions.

The purpose of this utility is to enable you to easily convert between rotation systems used by different modules.

Rotation Representations

These are the representations that PXC[M]Rotation supports:

•Quaternion – a 4D float vector PXC[M]Point4DF32 that represents a 3D rotation (w,x,y,z).
•Rotation Matrix – a 3x3 float rotation matrix.
•Euler Angles – a 3D float vector representing the angle of rotation around each axis (pitch/yaw/roll).
•Angle + Axis - rotation angle (in Radians) around an axis.

Creating a Rotation Instance

There are two steps for initializing a rotation instance:

1.Create the PXC[M]Rotation instance
2.Set the input representation of the rotation. You can use any of the following PXC[M]Rotation methods, depending on your input representation of the rotation:
oSetFromQuaternion  - set the input representation as the given quaternion.
oSetFromRotationMatrix - set the input representation as the given rotation matrix.
oSetFromEulerAngles - set the input representation as the given Euler angles.
oSetFromAngleAxis - set the input representation as the given angle and axis.
oSetFromSlerp - initialize the PXC[M]Rotation instance as the slerp of the two given PXC[M]Rotation instances and the “slerp factor”.  A slerp is the spherical linear interpolation between two rotations. The slerp factor indicates how much should each given rotation influences the final result.

Example 102 demonstrates how to create a rotation instance from a rotation matrix.

C++ Example 102: Create a Rotation Instance

// Create PXCsession instance

PXCSession* session = PXCSession::CreateInstance();

 

// Create PXCRotation instance

PXCRotation* rotation;

session->CreateImpl<PXCRotation>(&rotation);

 

// Set Rotation instance from 3x3 rotation matrix

pxcF32 rotationMatrix[3][3] = {{1,0,0}, {0,1,0}, {0,0,1}};

rotation->SetFromRotationMatrix(rotationMatrix);

C# Example 102: Create a Rotation Instance

Using the Rotation

Use the following methods to query a specific representation of the PXC[M]Rotation object, or to perform a new rotation:

•The QueryQuaternion, QueryRotationMatrix, QueryEulerAngles and QueryAngleAxis functions return the current rotation in the specified representation.
•The QueryRoll, QueryPitch and QueryYaw functions return the angle around the x-axis, y-axis and z-axis respectively.
•The Rotate function performs a new rotation relative to the current one. I.e. concatenates the current rotation and the given one.
•The RotateVector function returns the given vector rotated according to the current rotation.

When using the QueryEulerAngles function, there are a few different representations for the same rotation angle. The EulerOrder enumeration indicates the order in which to extract the Euler angles. This order matters (e.g. ROLL_PITCH_YAW != ROLL_YAW_PITCH), and will result in different output values, but they all correspond to the same rotation.

Example 103 demonstrates how to retrieve the rotation in different representations.

C++ Example 103: Retrieve Rotation In Different Representations

// Get the current rotation in the Euler representation

PXCPoint3DF32 eulerAngles = rotation->QueryEulerAngles(PXCRotation::PITCH_YAW_ROLL);

 

// Get the current Angle+Axis representation

PXCMRotation::AngleAxis angleAxis = rotation->QueryAngleAxis();

C# Example 103: Retrieve Rotation In Different Representations

// Get the current rotation in the Euler representation

PXCMPoint3DF32 eulerAngles = rotation.QueryEulerAngles(PXCMRotation.EulerOrder.PITCH_YAW_ROLL);

 

// Get the current Angle+Axis representation

PXCMRotation.AngleAxis angleAxis = rotation.QueryAngleAxis();

Java Example 103: Retrieve Rotation In Different Representations

Full Code Sample

Example 104 demonstrates the use of the rotation utility with the hand module.

C++ Example 104: Use Rotation with Hand

// Init

PXCSenseManager* senseManager = PXCSenseManager::CreateInstance();

 

// Hand Module Setup

senseManager->EnableHand();

PXCHandModule* handModule = senseManager->QueryHand();

PXCHandData* handDataOutput = handModule->CreateOutput();

 

// Creating PXCRotation instance

PXCRotation* rotation;

senseManager->QuerySession()->CreateImpl<PXCRotation>(&rotation);

 

// First Initializing the sense manager

if(senseManager->Init() >= PXC_STATUS_NO_ERROR) {        

   // Acquiring frames from input device

   while(senseManager->AcquireFrame(true) >= PXC_STATUS_NO_ERROR) {

       // Get current frame hand output

       handDataOutput->Update();

 

       if(handDataOutput->QueryNumberOfHands() > 0) {

           PXCHandData::IHand *hd;

           pxcStatus status=handDataOutput->QueryHandData(PXCHandData::ACCESS_ORDER_BY_TIME,0,hd);

           if (status>=PXC_STATUS_NO_ERROR && hd->HasTrackedJoints()) {

               PXCHandData::JointData jd;

               hd->QueryTrackedJoint(PXCHandData::JOINT_CENTER,jd);

                                 

               // Set Rotation to the orientation of the hand’s center

               PXCPoint4DF32 quaternion = jd.globalOrientation;

               rotation->SetFromQuaternion(quaternion);        

                                 

               // You can get the current rotation in other representations.

               PXCPoint3DF32 eulerAngles = rotation->QueryEulerAngles();

                 

               // You can rotate a vector with the current rotation.

               PXCPoint3DF32 vec = {1.f, 1.f, 1.f};

               PXCPoint3DF32 rotatedVec = rotation->RotateVector(vec);

           }        

       }

 

       // Finished working on current frame

       senseManager->ReleaseFrame();

   } // end while acquire frame

} // end if Init

 

//Release

rotation->Release();

handDataOutput->Release();

senseManager->Release();

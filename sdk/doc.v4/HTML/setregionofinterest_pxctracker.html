Parameters

 

roi

The region of interest rectangle.

Description

The SetRegionOfInterest function specifies a region of interest during tracking or map creation. Only pixels within the region are used for tracking or removal of cluttered background.

The region of interest is in respect to the color image space. If width=height=0, the region of interest is not used.

You can change the region of interest during tracking or map creation. The newly specified region of interest is effective on subsequent frames.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

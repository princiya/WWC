The PXC[M]Audio interface exposes the following member functions:

Member Functions

Description

AcquireAccess

Lock the audio buffer for exclusive access, with format conversion.

CopyAudio

Copy the audio buffer from another audio object.

QueryInfo

Return the AudioInfo structure.

QueryOption

Return the audio option flags.

QueryTimeStamp

Return the time stamp.

ReleaseAccess

Release the lock to the audio buffer.

SetOption

Set the audio option flags.

SetTimeStamp

Set the time stamp.

AddRef

Increase the reference counter of the instance.

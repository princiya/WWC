Configuration and Data Retrieval

 Top  Previous  Next

The hand tracking module exposes three main interfaces:

•The module interface (PXCFaceModule): This interface provides functions to create instances of the below two interfaces.
•The configuration interface (PXCFaceConfiguration): Use this interface to configure the face tracking module, including face detection, landmark detection, pose detection, and alert handling.
•The data retrieval interface (PXCFaceData): Use this interface to retrieve any available tracking and alert data.

The configuration interface PXCFaceConfiguration is a storage class that is, once created, independent of the face tracking module. Use the Update function to retrieve the most recent module configuration parameters and the ApplyChanges function to push the changes back to the face tracking module, as illustrated in Example 43.

Call the ApplyChanges function to apply any chances to the face tracking module.

C++ Example 43: Configure Face Tracking

// face is a PXCFaceModule instance

PXCFaceConfiguration *cfg=face->CreateActiveConfiguration();

 

// Set to enable all alerts

cfg->EnableAllAlerts();

 

// Apply changes

cfg->ApplyChanges();

 

// Clean up

cfg->Release();

C# Example 43: Configure Face Tracking

// face is a PXCMFaceModule instance

PXCMFaceConfiguration cfg=face.CreateActiveConfiguration();

 

// Set to enable all alerts

cfg.EnableAllAlerts();

 

// Apply changes

cfg.ApplyChanges();

 

// Clean up

cfg.Dispose();

Java Example 43: Configure Face Tracking

// face is a PXCMFaceModule instance

PXCMFaceConfiguration cfg=face.CreateActiveConfiguration();

 

// Set to enable all alerts

cfg.EnableAllAlerts();

 

// Apply changes

cfg.ApplyChanges();

 

// Clean up

cfg.close();

Similarly, the data retrieval interface PXCFaceData is a storage class as well. Use the Update function to retrieve the most recent processed data, as illustrated in Example 44.

C++ Example 44: Get Face Tracking Processed Data

// face is a PXCFaceModule instance

PXCFaceData *fdata=face->CreateOutput();

 

// Get the number of tracked faces

pxcI32 nfaces=fdata->QueryNumberOfDetectedFaces();

 

// Clean up

fdata->Release();

C# Example 44: Get Face Tracking Processed Data

// face is a PXCMFaceModule instance

PXCMFaceData fdata=face.CreateOutput();

 

// Get the number of tracked faces

Int32 nfaces=fdata.QueryNumberOfDetectedFaces();

 

// Clean up

fdata.Dispose();

Java Example 44: Get Face Tracking Processed Data

// face is a PXCMFaceModule instance

PXCMFaceData fdata=face.CreateOutput();

 

// Get the number of tracked faces

int nfaces=fdata.QueryNumberOfDetectedFaces();

 

// Clean up

fdata.close();

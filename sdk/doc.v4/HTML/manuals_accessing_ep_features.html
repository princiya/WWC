Accessing EP Features

 Top  Previous  Next

To use the Enhanced Photography features, create an instance of the PXC[M]EnhancedPhotography interface from PXC[M]Session. Then call the corresponding member functions. For example, call the DepthRefocus function for refocusing a picture, or the MeasureDistance function for measuring distance of two points in the picture. Example 95 shows how to perform refocusing given a focus point.

C++ Example 95: Photo Refocus

// session is a PXCSession instance

// Create an EP instance

PXCEnhancedPhotography *ep;

session->CreateImpl<PXCEnhancedPhotography>(&ep);

 

// Refocus the image.

// photo is a PXCPhoto instance.

PXCPointI32 focusPoint={ 20, 20 };

PXCPhoto *photo2=ep->DepthRefocus(photo, focusPoint);

 

// Do something with the processed photo2

...

 

// clean up

photo2->Release();

ep->Release();

C# Example 95: Photo Refocus

// session is a PXCMSession instance

// Create an EP instance

PXCMEnhancedPhotography ep;

session.CreateImpl<PXCEnhancedPhotography>(out ep);

 

// Refocus the image.

// photo is a PXCMPhoto instance.

PXCMPointI32 focusPoint=new PXCMPointI32(20, 20);

PXCMPhoto photo2=ep.DepthRefocus(photo, focusPoint);

 

// Do something with the processed photo2

...

 

// Clean up

photo2.Dispose();

ep.Dispose();

Parameters

 

blocking

A flag to indicate if the function should block or return immediately.

Description

The StreamFrames function is a helper function that implements the AcquireFrame/ReleaseFrame streaming loop in a separate thread.

If blocking=true, the function blocks until the streaming loop exits.

If blocking=false, the function returns immediately without blocking.

In both cases, the application can call the Close function to terminate at any time and release the resources.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

Remarks

If the application initiates the streaming loop through this function, the functions AcquireFrame and ReleaseFrame will return errors to indicate the duplicated invoking.

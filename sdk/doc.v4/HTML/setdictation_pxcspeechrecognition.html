Description

The SetDictation function sets the dictation mode.

Return Status

 

PXC[M]_STATUS_NO_ERROR

The function returned successfully.

Remarks

This function could block the execution for some time (a few seconds) for loading any language data the first time. It is recommended that the application mitigate this latency by some UI activities.

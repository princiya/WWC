The PXC[M]FaceData interface exposes the following member functions:

Feature

Member Function

Description

Alert Handling

QueryFiredAlertData

Return the fired alert of a given index.

QueryFiredAlertsNumber

Return the number of fired alert events.

QueryAlertNameByID

Return the fired alert event name.

IsAlertFired

Check if an alert is fired.

IsAlertFiredByFace

Check if an alert is fired by the specified face.

Face Tracking

QueryTimestamp

Return the time of detection.

QueryNumberOfDetectedFaces

Return the number of detected faces.

QueryFaceByID

Return the face data of the specified face.

QueryFaceByIndex

Return the face data by index.

QueryFaces

Return an array of all detected faces.

Face Recognition

QueryRecognitionModule

Return the database management instance.

Configuration

Update

Update the PXC[M]FaceData instance to be latest available output.

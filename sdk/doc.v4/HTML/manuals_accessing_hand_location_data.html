Accessing Hand Location Data

 Top  Previous  Next

You can use hand location data to map a hand’s action to a reaction in your application, and/or to represent the hand in your application’s display. The following details only describe the general hand location; if you need specific joint location and/or finger posture data, see Accessing Finger Data and Accessing Joint Data.

The hand location data does not require full-hand tracking, and is available in extremity tracking mode as well.

The IHand object contains the following methods for retrieving hand location data:

•QueryBoundingBoxImage – returns the location and dimensions of the tracked hand, represented by a 2D bounding box (defined in pixels).
•QueryMassCenterImage – returns the 2D center of mass of the hand in image space (in pixels).
•QueryMassCenterWorld – returns the 3D center of mass of the hand in world space (in meters).
•QueryExtremityPoint – returns the location of a specific hand extremity point. Extremities include the top-most, bottom-most, left-most, right-most, and closest points of the hand.

Description

The LandmarkPointSource structure describes the landmark point name.

 

Members

 

index

The index that this landmark belongs. See Figure 72 for illustration of all supported landmark points and their indexes.

Do not hard-code the index value as the supported number of points might change.

 

alias

The enumerated name of the landmark point. See the LandmarkType enumerator for definitions.

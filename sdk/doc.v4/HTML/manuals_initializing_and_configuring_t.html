Initializing and Configuring the Hand Module

 Top  Previous  Next

Before hand tracking can start, you must initialize PXC[M]SenseManager for hand tracking. Then you can optionally change its default configuration.

Example 61 demonstrates how to initialize PXC[M]SenseManager for hand tracking and how to configure the hand module.

C++ Example 61: Initialize SenseManager and the Hand Module

// Create an instance of the SenseManager

PXCSenseManager* sm=PXCSenseManager::CreateInstance();

 

// Enable hand tracking

sm->EnableHand();

 

// Get an instance of PXCHandModule

PXCHandModule *handModule=sm->QueryHand();

 

// Get an instance of PXCHandConfiguration

PXCHandConfiguration* handConfig=handModule->CreateActiveConfiguration();

 

// Make configuration changes and apply them

handConfig->EnableJointSpeed(…);

handConfig->SetTrackingBounds(…);

// … set other configuration options

handConfig->ApplyChanges(); // Changes only take effect when you call ApplyChanges

 

// Initialize the SenseManager pipeline

sm->Init();

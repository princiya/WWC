Description

The PXC[M]SenseManager helper interface provides a simple interface for some common application usage cases, that is, one or more processing modules (such as hand tracking, face analysis and voice recognition) connects directly to an I/O device (or a playback file).

The general procedure for using the PXC[M]SenseManager interface is as follows:

▪Configure: The application uses the EnableXXXX set of functions to configure the pipeline. For example, use the EnableStream function to specify a video stream.
▪Initialize: The application calls the Init function to initialize the pipeline.
▪Run the pipeline: The application calls the StreamFrames function, or the equivalent set of AcquireFrame/ReleaseFrame functions to loop frames among the pipeline components.
▪Handle events:  Optionally, the application can override the callback/delegate functions to handle events. See the Handler interface for details.

The application can use the CreateInstance function to create an instance of the PXC[M]SenseManager interface (or the CreateSenseManager function to create an instance from the session.)

Examples

Example 137 shows how to create a PXC[M]SenseManager instance and stream a color stream.

C++ Example 137: Configure and Stream a Color Stream

// Create a SenseManager instance

PXCSenseManager *sm=PXCSenseManager::CreateInstance();

 

// Enable the video stream

sm->EnableStream(PXCCapture::STREAM_TYPE_COLOR,0,0,0);

 

// Initialization

sm->Init();

 

// Stream data

while (sm->AcquireFrame(true)>=PXC_STATUS_NO_ERROR) {

 

   // Get the sample data

   PXCCapture::Sample *sample=sm->QuerySample();

   .... // process image sample->color

 

   // Resume next frame processing

   sm->ReleaseFrame();

}

 

// Clean up

sm->Release();

PXCAudioSourceService::QueryProfile

Syntax

C++

pxcStatus QueryProfile(pxcI32 pidx, AudioProfile *profile);

pxcStatus QueryProfile(AudioProfile *profile);

Parameters

 

pidx

The zero-based index to enumerate all available configurations.

 

profile

The audio device configuration, as described in the AudioProfile structure, to be returned.

Description

The QueryProfile function retrieves the specified audio device configuration parameters. If pidx is omitted, the function returns the current working configurations.

Return Status

 

PXC_STATUS_NO_ERROR

The function returned successfully.

 

PXC_STATUS_ITEM_UNAVAILABLE

The pidx value is out of range.

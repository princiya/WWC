Accessing Photo

 Top  Previous  Next

Most Enhanced Photography (EP) functions take a photo picture as input and post-process it. The photo instance can be from a Google* Depth File or from any preview image. The photo instance contains the following information:

•Color: The color image of the photo picture.
•Depth: The depth image of the photo picture.
•Calibration Data: Any auxiliary information required to map the depth image coordinates to the color image coordinates.

Creating Photo From Preview

Use the following steps to obtain a photo instance from any preview sample (see Example 92):

•Use the CreatePhoto function to create a photo image instance.
•Use the ImportFromPreviewSample function to initialize the photo content from the preview sample.

C++ Example 92: Take Snapshot using the SenseManager

void TakeSnapShot(void) {

   // Create a SenseManager instance

   PXCSenseManager *sm=PXCSenseManager::CreateInstance();

 

   // Create a photo instance

   PXCPhoto *photo=sm->QuerySession()->CreatePhoto();

 

   // Select the color and depth streams

   sm->EnableStream(PXCCapture::STREAM_TYPE_COLOR,320,240,30);

   sm->EnableStream(PXCCapture::STREAM_TYPE_DEPTH,320,240,30);

 

   // Initialize and Stream Samples

   sm->Init();

   for (;;) {

       // This function blocks until both samples are ready

       if (sm->AcquireFrame(true)<PXC_STATUS_NO_ERROR) break;

 

       // retrieve the photo picture

       PXCCapture::Sample *sample=sm->QuerySample();

 

       // Work on the photo with EP functions if this is the right snapshot

       photo->ImportFromPreviewSample(sample);

       ...

 

       // go fetching the next samples

       sm->ReleaseFrame();

   }

 

   // Close down

   photo->Release();

   sm->Release();

}

Creating Photo From File

Use the following steps to obtain a photo instance from a Google Depth File (see Example 93):

•Use the CreatePhoto function to create a photo image instance.
•Use the LoadXMP function to load the photo content from the file.

C++ Example 93: Read Photo From File

// session is the PXCSession instance.

// Create a photo instance

PXCPhoto *photo=session->CreatePhoto();

 

// Load the XMP file

photo->LoadXMP(L"C:/mywork/photo.xmp");

 

// Clean up

photo->Release();

C# Example 93: Read Photo From File

// session is the PXCMSession instance.

// Create a photo instance

PXCMPhoto photo=session.CreatePhoto();

 

// Load the XMP file

photo.LoadXMP("C:/mywork/photo.xmp");

 

// Clean up

photo.Dispose();

Saving Photo To File

Similarly, use the SaveXMP function to export to a Google Depth File. See Example 94.

C++ Example 94: Save Photo To File

// photo is a PXCPhoto instance

photo->SaveXMP(L"C:/mywork/photo.xmp");

C# Example 94: Save Photo To File

// photo is a PXCMPhoto instance

photo.SaveXMP("C:/mywork/photo.xmp");

Java Example 94: Save Photo To File

// photo is a PXCMPhoto instance

photo.SaveXMP("C:/mywork/photo.xmp");
